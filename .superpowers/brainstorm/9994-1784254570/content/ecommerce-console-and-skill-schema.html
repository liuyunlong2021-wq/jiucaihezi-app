Skill 如何“长出”少量参数

不是模型猜 UI，也不是 Skill 写网页。Skill 可选附一份声明：哪些事实需要用户选、可选项是什么、默认值是什么。

SKILL.md --- name: 电商商品图 description: ... ui: fields: - id: deliverable label: 交付目标 type: select options: [主图, 场景图, 详情页] - id: channel label: 发布位置 type: select optional: true ---
Skill Creator 创建或修改 Skill
↓
同一份 Skill 的 UI 声明
↓
现有动态字段渲染器显示控件
↓
值和用户原话一起送进创模式内核
边界：没有 UI 声明的 Skill 就是普通对话 Skill；不会被强行生成一堆参数。UI 声明只能描述输入，不能执行脚本、调用接口或绕过创模式工具审批。
