创建到使用的闭环

用户不需要懂 Schema。Skill Creator 在 Chat 里问清楚需求后，自动产出标准 Skill 和可选 UI 声明。

1. AI 协作记录“帮我做一个 Amazon 护肤详情页的电商 Skill。”
↓
2. Skill Creator澄清输入、交付物、必要选项，生成/评估 Skill。
↓
3. 兼容检查只有声明为 surface: ecommerce 且 UI 字段合法，才显示“加入电商工作台”。
↓
4. 自建页签新 Skill 出现在“自建”列表。点击使用，只为本轮创模式挂载它。
不变：同一个创模式会话、.raw 账本、热记忆、工具循环、三按钮审批、结果落到创作面板/画布。
不允许：Skill 自带 Vue 页面、任意脚本或新的请求通道。它只能声明输入控件，模型和工具仍由既有内核执行。
