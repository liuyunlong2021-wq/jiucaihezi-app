# Phase 0 - 手感基础设施 交付清单

**目标**：为 V8 画布建立高性能、可缩放的基础交互层。

## 已完成核心交付物

### 1. 全局交互冻结策略
- 文件：`composables/useCanvasInteractionFreeze.ts`
- 特性：
  - 支持嵌套交互（多个同时进行的 drag/resize）
  - 自动添加 `.canvas-interacting` class 到 body
  - 延迟恢复机制（防止闪烁）
  - 自动清理

### 2. 高性能 Resize 系统
- 文件：`composables/useNodeResize.ts`
- 特性：
  - 完全基于原生 PointerEvent + requestAnimationFrame
  - 拖拽过程中**零** Pinia / Vue 响应式更新
  - 结束时才批量提交最终尺寸
  - 自动集成全局冻结策略

### 3. 统一节点骨架
- 文件：`NodeFrame.vue`
- 特性：
  - 6 种角色色左边条
  - 完整支持 `collapsed` 状态（内容区不渲染）
  - 标准化底部执行条（▶ ■ ✕）
  - 内置 resize handle 事件
  - 数据属性便于调试和测试

### 3.1 高级行为组合
- 文件：`composables/useNodeBehavior.ts`
- 一站式组合 Freeze + Resize + 状态，提供更高级的节点行为封装。

### 4. 性能基准工具
- 文件：
  - `utils/performanceBenchmark.ts`
  - `utils/simulateHeavyCanvas.ts`
  - `examples/runBenchmark.ts`
- 特性：
  - 可测量 Jank（丢帧情况）
  - 内置 30 节点重负载模拟器
  - 一键运行基准（window.runV8Phase0Benchmark）

### 5. 测试覆盖（基础）
- `useCanvasInteractionFreeze.test.ts`
- `useNodeResize.test.ts`
- `performanceBenchmark.test.ts`

### 6. 使用与演示
- `examples/NodeDemo.vue`
- `examples/NodeDemoFull.vue`
- `USAGE.md`
- `README-Phase0.md`

### 7. 样式系统
- `styles/canvas-freeze.css` （核心冻结策略样式）
- `styles/node-base.css`
- `styles/freeze-enhance.css` （针对 VueFlow 的额外性能优化）

### 8. 便捷入口
- `index.ts`（自动加载样式 + 完整导出）
- `activate.ts`（一键激活开发模式）
- `composables/useActivateCanvas.ts`
