# 角色档案模板

> 逐集成稿专用。首次进入时从梗概抽取初版，**只填梗概里明确或可合理推断的信息**，没写到的留空，随每集定稿逐渐补全。档案是"长出来的"，不是一次写全的。

## 档案结构

```markdown
## 角色名

### 基础信息
- 性别 / 年龄 / 外貌 / 身高 / 形象气质：[首次能推断的填，不能的留空]
- 当前身份：[梗概里的身份]

### 核心
- 极致维度：[自由/正义/复仇/爱情/权力/知识/生存/荣誉/贪婪/守护...]（8-10/10）
- 内核动机：[一句话——他到底要什么]
- 红线：[不可触碰的底线，触碰必冲突]

### 性格
- 性格特点：[3-5 个关键词]
- 性格反差：[如外冷内热、笑面虎]
- 行为模式：[压力下的典型反应]

### 关系
- 与[主角/某角色]：[利益/情感/冲突关系]

### 背景
- 家庭 / 学识 / 经历：[梗概透露的部分]

### 成长轨迹（随定稿更新）
- 第X集：[本集暴露的新信息 / 性格转折 / 身份揭示 / 关系变化]
```

## 极致维度参考表

| 维度 | 红线（触碰必冲突） |
|------|---------------------|
| 自由 | 被束缚、被命令 |
| 正义 | 目睹不公、被冤枉 |
| 复仇 | 仇人逍遥、被羞辱 |
| 爱情 | 失去爱人、被背叛 |
| 权力 | 失去地位、被轻视 |
| 知识 | 真相被隐瞒 |
| 生存 | 面临死亡威胁 |
| 荣誉 | 名声被毁 |
| 贪婪 | 财富被夺 |
| 守护 | 要保护的人受威胁 |

## 使用要点

- **首次抽取**：通读全剧梗概后，给主角和主要配角各建一份。极致维度和内核动机要从梗概里人物的反复行为去归纳，归纳不出来的标"待定"，别硬编。
- **逐集回写**：每集定稿后，把本集暴露的新信息补进对应档案的「成长轨迹」，并在上方字段同步修正。这是长剧防失忆、防人设崩的命脉。
- **防人设崩**：每集成稿后对照档案，检查角色行为是否符合其极致维度和红线。若梗概要求的行为与已建档案冲突，**先提醒用户**，可能是档案归纳错了，也可能是梗概里的转折点——由用户判断，不擅自改。
