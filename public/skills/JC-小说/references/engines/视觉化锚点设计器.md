# SKILL-02: 视觉化锚点设计器 (Visual Anchors)

> 电影和小说通过"看"来传达信息。把抽象的形容词转化为具体的物理动作。

---

## 核心功能

将心理状态（"生气"、"焦虑"、"轻视"）转化为物理动词（Action Verbs）和业务行为（Business）。

---

## 核心技法

### 1. 物理动作设计 (Physical Business)

* **定义**：在对话过程中，角色手头正在做的事情。
* **拒绝**：两个人大头特写（Talking Heads），干坐着聊天。
* **替代方案**：
  * 修车、做饭、洗碗、擦枪、打高尔夫、给猫梳毛、拆弹、缝伤口。
* **作用**：
  * 增加画面活力。
  * 提供潜台词（用力切牛排 = 愤怒）。
  * 控制节奏（停下动作 = 重点强调）。

### 2. "演"出来 (Show, Don't Tell)

* **转化表**：

| 抽象描述 | 视觉化处理 (SHOW) |
| :--- | :--- |
| 他很生气 | 他将手中的铅笔却成了两截。 |
| 她很紧张 | 她不停地撕扯啤酒瓶上的标签。 |
| 他看不起他 | 他一边说话，一边专心地给指甲抛光，一眼都没抬。 |
| 房间很脏 | 发黄的披萨盒堆在墙角，一只蟑螂爬过遥控器。 |
| 他们关系疏远 | 两人坐在长餐桌的两端，进食声音在空荡的房间里回响。 |

### 3. 环境互动 (Environmental Interaction)

* **利用道具**：不要让角色空手。利用周围环境的道具来表达情绪。
  * 关窗 = 拒绝沟通。
  * 开灯 = 揭示真相/开始审视。
  * 倒水溢出 =内心慌乱。

---

## 执行指令

**Input**:
"老板对员工很不满，正在训话。"

**Process**:

1. **Identify Abstract**: "不满"、"训话"。
2. **Design Business**: 老板正在喂鱼。
3. **Visualization**: 老板背对着员工，慢慢撒鱼食。员工站在背后。老板说："这条鱼要是再不抢食，就该捞出来扔了。"（潜台词：你也一样）。

**Output**:
描述具体的物理动作链条。
