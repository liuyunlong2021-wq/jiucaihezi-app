# 催化提取

## 模型何时使用

模型判断小说、剧本或参考作品中存在值得复用的节奏模式，且用户当前创作需要更快检索“类似情境如何推进”时使用。它提取叙事结构，不复制具体情节和表达。

## 输出标准

- 每个被提取片段记录：情境、冲突、人物关系、推进动作、反转/爽点、钩子和来源。
- 生成按情境标签组织的节拍索引，例如“身份暴露”“故人重逢”“资源争夺”，每项链接到具体片段。
- 索引必须让模型先按当前情境找到节拍，再按需阅读细节；必要时更新创作项目的参考入口。

示例：

```markdown
## 冲突：身份暴露
- [[催化/第8章]]：被认出 -> 否认 -> 证据逼近 -> 被迫承认 -> 试探立场
```

## 安全边界

不抄原文，不把参考作品人物/设定当作当前项目事实；没有明确的可迁移价值时不为了完整性全量拆书。
