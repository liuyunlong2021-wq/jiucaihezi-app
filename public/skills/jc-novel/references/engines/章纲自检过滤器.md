# SKILL-05: 章纲自检过滤器 (Chapter Detox)

> 质量控制的最后一道关卡。

---

## 核心功能

检查章纲/场景卡片是否符合 40 卡板、符号系统和晚进早出规则。

---

## 检查清单 (The Checklist)

### 1. 极性单一检查 (Polarity Stagnation)

* **Check**: 扫描所有卡片。是否有 `+ -> +` 或 `- -> -` 的情况？
* **Rule**: 凡是极性未发生反转的场景，要么重写（加入转折），要么删除（无效场景）。

### 2. 单向冲突检查 (One-Way Conflict)

* **Check**: 是否只有 `>` 没有 `<`？（主角太顺了）。或者只有 `<` 没有 `>`？（主角太被动）。
* **Rule**: 必须是双向力。

### 3. 晚进早出检查 (Trim Check)

* **Check**: 场景描述是否包含了"进门"、"寒暄"、"送客"？
* **Rule**: 删掉。直接从冲突开始，冲突结束立刻切。

### 4. 40卡限额检查 (Quota Check)

* **Check**: 总卡片数是否超过 40？
* **Rule**: 如果超过，必须精简。长剧本会稀释张力。

---

## 执行指令

**Input**:
完整章纲。

**Process**:

1. Run Checklist 1-4.
2. **Report**: 标记有问题的卡片（如：Card #12 Polarity Fail）。
3. **Suggest**: 给出修改建议。

**Output**:

* **自检报告**: ...
