# 导演式交互分镜（剧本 → 导演风格 → 置景+光影 → 分镜）

## Overview

输入剧本、输出置景 + 光影 + 分镜。全程 A/B/C/D 选项交互，模型只推荐不拍板。不输出任何 JSON。

分镜格式：`集-场-镜 | 景别 | 角度 | 运动 | video_prompt`

核心原则：**每一镜的景别、角度、运动、角色表演、置景方向，全部跟随用户选定的导演作品**。导演切镜碎则分镜碎（如徐克一场对打几十镜），导演用长镜则分镜少（如落水狗几镜打完）。

## 剧本输入格式

用户粘贴的剧本遵循标准格式，直接从中解析集号、场号、时间、场景、人物：

```
#集
## 场X-Y
时间：[日/夜/黄昏]（+雨/雪/风）
场景：[室内/室外]+[具体地点]
人物：角色A 角色B （次要人物用甲乙丙标注）

△ 场景/动作描述

角色名（情绪）："台词"
△ 动作描述
```

- `X` = 集号，`Y` = 场内序号。解析后映射为 `X-Y`
- 若剧本无 `#集`，默认集号=1
- `## 场X-Y` 后面的 `X-Y` 就是最终分镜使用的场号

## 核心规则

1. A/B/C/D 选项推进，不替用户决定
2. 风格必须「导演 + 作品」，不能是空词
3. 光影、置景、景别、角度、运动全部跟随选定作品
4. **切镜密度跟导演走**：动作戏参考导演真实作品里同类场景的镜头数——徐克碎切、王家卫慢推、落水狗几镜一场——不套通用模板
5. **代词规则**：每个镜头第一次出现某角色必须用角色全名；同一镜头内可用代词；**严禁跨镜头用代词**——上一镜有小红，下一镜写「她举起镰刀」不行，必须写「小红举起镰刀」
6. 零 JSON 输出

## 跨轮次记忆

多轮对话时每轮先回顾前文已选结果（风格、各场光影、已完成分镜），确认「继续 / 重来」。

## 阶段 0：解析剧本

1. 读取剧本，按标准格式解析出集号、场号、时间、场景、人物、△动作、台词
2. 若剧本无标准格式头，按「场 / 景 / △ / 空行」拆分，默认集=1、场=1起自增
3. 选项确认拆分：**A 接受 / B 重新拆分 / C 直接进风格推荐**

## 阶段 0.5：媒介分类

- **A 动漫** → 风格从 `references/anime-style-bank.md`
- **B 真人** → 风格从 `references/director-style-bank.md`

## 阶段 1：推荐导演作品风格（A/B/C/D）

每个推荐含：
- 导演 + 作品名 + 为什么适合
- 画幅：9:16 或 16:9
- 风格基调：`[布光] + [色调] + [反差] + [质感]`
- 节奏与切镜密度：具体到秒/镜、碎切或长镜、同类场景镜数参考（如「对打戏 30-50 镜」「对峙戏 3-5 镜」）
- 置景美学：该导演场景陈设方向（材质/年代感/空间密度/标志物）
- 景别习惯：偏好哪些景别组合（如「中近景→特写→插道具特写」或「固定远景极少近切」）
- 镜头运动习惯：推拉摇移跟的偏好（如「缓慢推轨少摇镜」「肩扛手持晃动」）
- 常用角度与高度：俯/仰/平的偏好（如「大量平视、偶尔仰拍英雄」）

**3 个选项必须覆盖不同方向**（情绪作者型 / 节奏商业型 / 写实纪录型），让用户有取舍。

不允许：抽象词、无具体作品、同一导演不同作品当三个选项。

选 A/B/C → 锁定风格进阶段 2。选 D → 收敛到「导演 + 作品」才能继续。

## 阶段 2：逐场置景 + 光影推荐（A/B/C/D）

每场分两步：

### 2a. 场景陈设

基于剧本的场景描述 + 选定导演的**置景美学**，写一段场景陈设。目的是让 LLM 立刻知道这是什么场景、触发训练数据中的视觉关联。写具体材质、物件、空间质感，不写抽象形容词。

示例（王家卫《花样年华》调性）：
> 1960年代香港旧公寓卧室——窄小空间堆满旧物，铁架上下铺花布床单，窗边木桌泡面盒与烟灰缸，墙角发黄空调机嗡嗡响，门框挂半透明塑料珠帘，走廊尽头公共洗手间荧光灯管频闪投进青白冷光。

置景也给 D 自定义：若用户不满意置景方向，可描述想要的陈设感觉，模型据此改写。

### 2b. 光影方案

基于选定作品的光影语言，给 3 个光影方案（A/B/C）+ D 自定义：

- 短名 + 光影设计 + 情绪 + 为什么适合

**推荐路径**：所有场次的置景+光影一次性列出，用户逐场分别确认置景和光影（各可独立选 D）。

## 阶段 3：分镜脚本

### 格式

每镜一行，五列管道分隔：

```
集-场-镜 | 景别 | 角度 | 运动 | prompt
```

- **集-场-镜**：纯数字，横杠分隔。集号场号从剧本解析，镜号每场从1自增
- **景别**：远景 / 全景 / 中景 / 近景 / 特写 / 大特写（跟随导演景别习惯）
- **角度**：平视 / 俯拍 / 仰拍（跟随导演角度习惯），可加高度细化：
  - `平·眼` 平视眼平 | `平·低` 平视低机位
  - `俯·高` 俯拍高机位 | `俯·眼` 俯拍眼平（少见）
  - `仰·低` 仰拍低机位 | `仰·眼` 仰拍眼平（少见）
- **运动**：固定 / 推 / 拉 / 摇 / 移 / 跟 / 手持 / 升降（跟随导演运动习惯）
- **prompt**：`[角色名+动作] + [镜头指令] + [气氛/光影] + [音效]`

示例：
```
1-1-1 | 中景 | 平·眼 | 固定 | 小红猛然转身看向门口，camera not moving，门口尘光散射，sharp impact sound
1-1-2 | 近景 | 平·眼 | 慢推 | 小红瞳孔收缩，小红的手颤抖着摸向腰间镰刀，camera slowly pushes in，暖橘台灯在她侧脸投下金边，tense low ambient hum
1-1-3 | 特写 | 平·眼 | 固定 | 小红直视镜头，小红冷淡的说：你到底是谁？，camera not moving，quiet room tone
```

### prompt 规则

1. 格式：`[动作动词] + [镜头指令] + [气氛/光影] + [情绪/音效]`
2. 内嵌该场的**光影设计**与**风格基调**作为气氛锚点
3. 不重复首帧已有内容——只写动作、镜头运动、气氛、音效
4. 锁死镜头用 `camera not moving`
5. 强图短提示（1-8词），需影调才长（20-60词）
6. 中文对白保留原台词

### 代词规则（强制）

- 每镜中角色**首次出现**必须用全名（如「小红」「张三」）
- 同一镜内再次提到可用「她」「他」
- **严禁跨镜代词**：镜3写了「小红举起镰刀」，镜4不能写「她砍下」→ 必须写「小红砍下」
- 每镜独立，当作观众不知道上一镜有谁

### 切镜密度（强制）

**不套通用模板**。参照选定导演真实作品中同类场景的镜头数：

- 徐克《新龙门客栈》对打 → 碎切，30-50镜
- 昆汀《落水狗》对峙 → 长镜，3-5镜
- 王家卫双人戏 → 慢推，5-8镜
- 温子仁恐怖 reveal → 中快，8-15镜

分镜数量和节奏必须对得上导演的真实风格。

### 审阅输出

每场分镜先写置景+光影头，再列分镜：

```
## 1-1 · 小泽公寓卧室
**置景**：1960年代香港旧公寓——窄小空间铁架上下铺花布床单，窗边木桌堆满泡面盒烟灰缸，墙角发黄空调机嗡嗡响，门框半透明塑料珠帘，走廊尽头荧光灯管频闪投进青白冷光。
**光影**：暖橘台灯 + 青蓝边（窗边冷蓝逆光 + 室内暖橘主光 + 中高反差 + 胶片颗粒）

1-1-1 | 中景 | 平·眼 | 固定 | 小红猛然转身看向门口，camera not moving，门口尘光散射，sharp impact sound
1-1-2 | 近景 | 平·眼 | 慢推 | 小红瞳孔收缩，小红的手颤抖着摸向腰间镰刀，camera slowly pushes in，暖橘台灯在小红侧脸投下金边，tense low ambient hum
1-1-3 | 特写 | 平·眼 | 固定 | 小红直视镜头，小红冷淡的说：你到底是谁？，camera not moving，quiet room tone
```

## 最终输出

全部场次完成后，汇总输出 `analysis/shot-design.md`（纯文本，无 JSON），格式与审阅输出一致：

```
# 项目名 · 导演式分镜
## 风格：王家卫《花样年华》| 画幅 9:16

---

## 1-1 · 小泽公寓卧室
**置景**：1960年代香港旧公寓——窄小空间铁架上下铺花布床单...
```
