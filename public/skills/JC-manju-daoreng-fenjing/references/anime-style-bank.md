# 动漫作品参考库（Anime Style Bank）

本库是 `JC-manju-daoreng-fenjing` 在**阶段 0.5 选 A 动漫**时使用的风格参考库，与 `director-style-bank.md`（真人库）字段格式一致。

- 动漫模式字段映射：`director` 填「监督 / 动画公司 / 原作」，`work` 填具体动画作品（或美影厂的「风格 + 代表作」）。
- 模型应基于真实动画知识生成推荐，**优先用库内已校准的格式模板**。
- 若用户指定作品不在库内，模型按该作品真实上色 / 演出 / 节奏特征生成，不必拘泥本库。
- 每个条目给出可直接复用的 `style_design`（含上色基调与光影）与 `pacing_design`（分镜节奏与演出张力）。
- 美影厂条目额外列「技法要点」与「变体」，方便阶段 1 把同一厂的不同美术片拆成差异化选项。

字段约定：
- `style_design` 吸收该作品的**标志性上色与光影语言**，格式 `[上色/布光] + [色调] + [反差] + [质感]`。
- `pacing_design` 反映该作品的**分镜节奏与切镜密度**（秒/镜、碎切或长镜），分镜阶段直接用于控制镜头数量。
- `置景美学` 该作品的场景陈设方向——建筑风格、时代质感、标志物，用于阶段 2 写场景陈设。
- `景别习惯` 该作品偏好哪些景别组合，分镜阶段直接用于决定每镜景别。

## 美影厂美术片（上海美术电影制片厂）

> 美影厂是中国动画学派代表，技法与日系 cel 完全不同。下面按「一种技法 = 一个可选项」拆开，方便阶段 1 给出差异化选项。竖屏短剧可统一改为 9:16；原始比例多为 4:3 / 1.37:1。

## 使用提示

阶段 1（动漫侧）推荐时，同样从以上方向各取一个（如「热血明亮 + 酷炫速度 + 美影厂工笔重彩」），保证 A/B/C 覆盖不同方向，让用户有明显取舍。动漫侧也必须具体到作品/风格，不能只给「日系」「热血」空词：
- 现代动画：`director` 写监督 / 动画公司 / 原作，`work` 写具体作品（如《火影忍者》《凡人修仙传》）。
- 美影厂：`work` 写「风格 + 代表作」（如《大闹天宫》工笔重彩 / 《九色鹿》敦煌壁画），让用户清楚选的是哪一种民族美术片技法。
