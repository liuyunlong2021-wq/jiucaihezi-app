# 导演作品参考库（Director Style Bank）

本库是 `JC-manju-daoreng-fenjing` 的**格式与一致性锚点**，不是封闭清单。

- 模型应基于真实电影知识生成推荐，**优先用库内已校准的格式模板**。
- 若用户指定作品不在库内，模型按该作品真实光影 / 节奏 / 机位特征生成，不必拘泥本库。
- 每个条目给出可直接复用的 `style_design` 与 `pacing_design` 模板，并提供「适用剧本类型」帮助阶段 1 做方向覆盖。

字段约定：
- `style_design` 吸收该作品**标志性光影语言**，格式 `[布光] + [色调] + [反差] + [质感]`。
- `pacing_design` 反映该作品**真实节奏与切镜密度**（秒/镜、碎切或长镜），分镜阶段直接用于控制镜头数量。
- `置景美学` 该导演标志性场景陈设方向——材质、年代感、空间密度、标志物，用于阶段 2 写场景陈设。
- `景别习惯` 该导演偏好哪些景别组合，分镜阶段直接用于决定每镜景别。

## 使用提示

阶段 1 推荐时，从以上方向各取一个，保证 A/B/C 覆盖不同方向（如「情绪作者型 + 写实温情型 + 冷峻惊悚型」），让用户有明显取舍。若剧本明显偏某类，仍应给出差异化的三个选项，而非三个近亲。
