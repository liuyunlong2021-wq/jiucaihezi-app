# 对话记录 Wiki 标准

## 判断指纹

- 多轮交替：`用户：`/`AI：` 或 `User:`/`Assistant:` 或时间戳分隔
- 回合数 > 5
- 有明显的问答/讨论模式
- 非剧本/小说格式

## 标准 Wiki 目录树

```
wiki/
├── 对话记录/
│   ├── _index.md              # 全量对话索引
│   ├── {主题1}/
│   │   ├── _index.md
│   │   ├── 对话1.md
│   │   └── ...
│   ├── {主题2}/
│   └── ...
├── （如对话中产生了创作内容，按对应标准追加目录）
└── index.md
```

## 各文件内容要求

### 对话记录/_index.md

```markdown
# 对话记录索引

## 统计
- 总对话数：X 条
- 时间跨度：YYYY-MM-DD ~ YYYY-MM-DD
- 主题分类：N 个

## 目录
- [[主题1/_index]]
- [[主题2/_index]]
```

### 对话记录/{主题}/对话N.md

```markdown
---
date: YYYY-MM-DD
topic: 主题标签
---

# 对话标题（提炼关键议题）

## 关键结论
- 结论 1
- 结论 2

## 相关
- [[相关笔记]]
- [[相关角色]]
```

## 分类规则

基于对话实际内容生成 3-4 个分类方案（A/B/C/D），用户选择：

| 方案 | 适用场景 |
|------|---------|
| 按主题分类 | 对话有 3-6 个明显主题聚类 |
| 按阶段分类 | 对话有明显先后顺序（策划→执行→复盘） |
| 按时间归档 | 对话跨度大且无明显聚类 |

## 读取时机

SKILL.md 阶段 1 匹配到对话记录 → 阶段 2 读本文件 → 按目录树和内容要求执行转换。
