# 漫剧/短剧 Wiki 标准

## 判断指纹

- 场次标记：`第X场`、`Scene X`、`【第X场】`
- 动作标记：`△`、`▲`、`【动作】`
- 对白格式：`角色名：台词`（非引号对话）
- 元数据：集号、场号、时间、场景、人物列表

## 标准 Wiki 目录树

```
wiki/
├── 角色/
│   ├── _index.md           # 角色总览表（名称/身份/关系/首次出场）
│   ├── 主角名.md            # 每个角色一个文件
│   └── ...
├── 世界/
│   ├── 世界观.md            # 时代/地点/规则/势力
│   └── 调性.md              # 风格/节奏/爽点类型
├── 剧本/
│   ├── 大纲.md              # 全剧梗概
│   ├── 第1场.md             # 逐场拆分
│   ├── 第2场.md
│   └── ...
├── 场景/
│   ├── _index.md            # 场景总览
│   ├── 场景名.md            # 每个场景一个文件（地点/视觉/道具）
│   └── ...
├── 道具/
│   ├── _index.md
│   └── ...
└── index.md                 # 全库目录 + 双链
```

## 各文件内容要求

### 角色/角色名.md

壳文件，初始化只写标题 + 出处标注：

```markdown
# 角色名

> 待摄入。来源：第 X 场首次出场。
```

内容由记忆压缩模式四填充。

### 世界/世界观.md

```markdown
# 世界观

## 时代背景
## 地点
## 规则/设定
## 势力分布
```

### 剧本/第X场.md

```markdown
# 第X场

- 时间：
- 场景：[[场景名]]
- 人物：[[角色A]]、[[角色B]]

## 正文
（完整场次内容，保留原格式）
```

### 剧本/大纲.md

按 SKILL.md 阶段 2 的大纲生成原则，四档优先级取来源。不硬性规定字数。标注来源。

## 拆分规则

1. 按场次标记拆分，每场一个 .md，原文保留，不改一字
2. 角色首次出场创建壳文件（标题 + 出处标注）
3. 场景首次出现创建壳文件
4. 大纲、角色小传、双链、hot.md 归 JC-Wiki-记忆压缩
5. `_index.md` 自动汇总

## 读取时机

SKILL.md 阶段 1 匹配到漫剧/短剧 → 阶段 2 读本文件 → 按目录树和内容要求执行转换。
