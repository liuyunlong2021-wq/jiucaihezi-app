# 小说 Wiki 标准

## 判断指纹

- 章节标记：`第X章`、`Chapter X`、`第X卷`
- 叙事体为主（非对白体占比 > 60%）
- 人物对话使用引号（`「」`、`""`、`''`）而非剧本格式
- 有连续的叙述段落

## 标准 Wiki 目录树

```
wiki/
├── 角色/
│   ├── _index.md           # 角色总览表
│   ├── 主角名.md
│   └── ...
├── 世界/
│   ├── 世界观.md            # 时代/地点/修炼体系/势力
│   ├── 设定集.md            # 功法/法宝/种族等专有设定
│   └── 地图.md              # 可选：地理描述
├── 剧情/
│   ├── 大纲.md              # 全书梗概
│   ├── 细纲.md              # 可选：分卷/分阶段纲要
│   ├── 第1章.md             # 逐章拆分
│   ├── 第2章.md
│   └── ...
├── 伏笔/
│   └── 伏笔管理.md          # 已埋/已收/待收
└── index.md
```

## 各文件内容要求

### 角色/角色名.md

壳文件，初始化只写标题 + 出处标注：

```markdown
# 角色名

> 待摄入。来源：第 X 章首次出场。
```

内容由记忆压缩模式四填充。

### 世界/世界观.md

```markdown
# 世界观

## 时代背景
## 地理
## 修炼体系/能力体系
## 势力分布
## 关键设定
```

### 剧情/第X章.md

```markdown
# 第X章：章节标题（如有）

## 概要
（200 字以内）

## 关键事件
- 事件 1
- 事件 2

## 出场角色
- [[角色A]]
- [[角色B]]

## 伏笔
- 埋了：[描述]
- 收了：[描述]

## 正文
（完整章节内容，保留原格式）
```

### 剧情/大纲.md

按 SKILL.md 阶段 2 的大纲生成原则，四档优先级取来源。不硬性规定字数。标注来源。

## 拆分规则

1. 按章节标记拆分，每章一个 .md，原文保留，不改一字
2. 角色首次出场创建壳文件（标题 + 出处标注）
3. 大纲、角色小传、双链、hot.md 归 JC-Wiki-记忆压缩
4. `_index.md` 自动汇总

## 读取时机

SKILL.md 阶段 1 匹配到小说 → 阶段 2 读本文件 → 按目录树和内容要求执行转换。
