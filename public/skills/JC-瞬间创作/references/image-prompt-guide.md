# 图片提示词写作指南

> 什么时候读：用户说「帮我写提示词」「我不会写」或提示词太短/模糊（< 15 字无画面细节）时。

## 输出格式

**一段中文自然语言，不分行不编号。** 按以下结构组织：

```
主体 + 场景 + 光影 + 构图 + 风格 + 画质 + 情绪
```

## 各维度要点

| 维度 | 关键词示例 |
|------|-----------|
| **主体** | 人物/动物/物品的外貌、服装、姿态、表情 |
| **场景** | 室内/室外、城市/自然、时间（黄昏/清晨/夜晚）、天气 |
| **光影** | 逆光/侧光/柔光/硬光、黄金时刻、霓虹灯、窗户光 |
| **构图** | 特写/中景/全景、浅景深、三分法、对称、俯拍/仰拍 |
| **风格** | 写实/油画/水彩/赛博朋克/日系/胶片/极简/插画 |
| **画质** | 4K/8K、超写实、HDR、电影级 |
| **情绪** | 温暖/冷峻/孤独/欢快/神秘/宁静 |

## 示例

```
弱提示词:
"生成一只猫"

增强后:
"一只橘色短毛猫慵懒地趴在木质窗台上，午后阳光从百叶窗斜射进来形成条纹光影，浅景深虚化窗外绿植，温暖的日系胶片色调，4K 超写实，宁静治愈"
```

```
弱提示词:
"画一个未来城市"

增强后:
"赛博朋克风格的未来都市夜景，高耸的霓虹摩天楼群笼罩在雨雾中，紫蓝色调主导画面，飞行汽车拖曳光轨穿过楼间，广角仰拍强化建筑压迫感，电影级渲染，神秘而震撼"
```

```
弱提示词:
"帮我画个美女"

增强后:
"东方女性半身肖像，黑色长发自然垂落，暖调侧光从左侧打来勾勒面部轮廓，深绿色背景虚化，浅景深聚焦眼神，写实油画质感，眼神中透着坚定与温柔"
```

## 原则

- 一段话，不分行
- 中文优先，可夹杂少量专业英文术语（bokeh, HDR, cyberpunk 等）
- 积极具体：写「暖黄灯光」不写「不要冷光」
- 每次只生成 1 版，用户不满意再改
