## 二、LLM 大语言模型（Chat 聊天区用）

聊天区顶部有个模型选择器，点开就能换。模型列表从韭菜盒子云端自动加载，主流模型都有：

**Claude 系**：Opus 4.6 / 4.7 / 4.8、Sonnet 4.6 / 5、Haiku 4.5、Fable 5

**GPT 系**：5.4 / 5.4-mini / 5.5 / 5.6-luna / 5.6-sol / 5.6-terra

**DeepSeek 系**：V4 Pro / V4 Flash

**Gemini 系**：3.1 Pro / 3.1 Flash Lite / 3.5 Flash

**Grok 系**：4.5 / 4.1 Fast

**国产**：Qwen 3.7 Plus / 3.7 Max、GLM-5.2、Doubao Seed 系列、Doubao Seed 2 系列、腾讯 HY3

**开源**：GPT-OSS 120B、Gemma 4 31B

**搜索**：Jina Search

### 文/武模式

- **文模式**：便宜模型，简单任务
- **武模式**：强模型（Claude/GPT），复杂任务

默认模型是 **Claude Sonnet 4.6**。第一次用不用换，直接聊就行。

## 三、韭菜盒子 API（自己搭客户端用）

不光能在韭菜盒子 App 里用这些模型——你还可以用韭菜盒子的 API，在任何 AI 客户端里调模型。

### 接口地址

```
https://api.jiucaihezi.studio
```

### 怎么用？

1. 去 [https://api.jiucaihezi.studio/keys](https://api.jiucaihezi.studio/keys) 生成密钥
2. 打开任意 AI 客户端：
   - **终端类**：Claude Code、CodeX、小龙虾、爱马仕、Cursor Switch
   - **桌面类**：ChatBox、Cherry Studio、NextChat
3. 填入接口地址 `https://api.jiucaihezi.studio` 和密钥
4. 选模型，开始用

### 创建密钥怎么选分组？

- **新手**：选「自动分组」，不用管细节
- **老手**：去[模型广场](https://api.jiucaihezi.studio/pricing)看，选特定分组
- **生图/生视频**：只有「1倍分组」和「自动分组」有

### 怎么收费？

按**倍率**算。比如选 0.3 倍率，就是 **0.3 元人民币 = 1 美金 Token 购买力**。所有模型的输入输出定价跟官方一样，以美金结算。

去 [https://api.jiucaihezi.studio/pricing](https://api.jiucaihezi.studio/pricing) 看模型分组和价格。

### 查花费 & 充值

- 花费记录：[https://api.jiucaihezi.studio/usage-logs/common](https://api.jiucaihezi.studio/usage-logs/common)
- 充值：[https://api.jiucaihezi.studio/wallet](https://api.jiucaihezi.studio/wallet)

> ⚠️ 密钥 = 账户密码，别分享给别人。

## 常见问题

### 不知道选哪个？
生图用 GPT Image 2。视频用 Seedance 2.0 Fast。聊天用 Claude Sonnet 4.6（默认）。都不出错。

### 怎么知道价格？
创作面板选模型时显示价格。API 去 [pricing](https://api.jiucaihezi.studio/pricing) 看倍率和分组。

### API 和 App 里的模型一样吗？
一样。App 聊天区用的就是 API 的 LLM 模型，创作面板是媒体模型。

### API Key 在哪生成？
[https://api.jiucaihezi.studio/keys](https://api.jiucaihezi.studio/keys)，选分组后生成。别分享给他人。
