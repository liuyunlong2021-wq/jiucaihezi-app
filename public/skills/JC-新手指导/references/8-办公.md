# 办公：用韭菜盒子跑开源项目 + PPT + 格式转换

> 韭菜盒子不只是创作工具——能装 GitHub 开源项目，帮你处理 PPT、Excel、PDF、Word 各种办公杂活。

---

## 一、安装 GitHub 开源项目

韭菜盒子能安装运行 GitHub 上的开源项目。不需要你自己配环境、装依赖——AI 帮你搞定。

### 怎么装？

在 Chat 区说：

```
帮我安装这个项目：https://github.com/XXX/XXX
```

AI 会自动：
1. 克隆代码到你的工作区
2. 读 README 分析项目怎么跑
3. 装依赖、配置环境
4. 告诉你安装好了，可以用了

### 推荐的高星办公项目

| 项目 | GitHub Stars | 能干啥 |
|------|-------------|--------|
| **Stirling PDF** | 60k+ | PDF 全能工具箱：合并/拆分/压缩/OCR/格式转换/加密/水印，网页版操作 |
| **Docling** | 30k+ | IBM 出品，PDF/DOCX/PPT/XLSX/图片 → Markdown/JSON，AI 文档解析 |
| **Marker** | 40k+ | PDF → Markdown 极速转换，表格和公式都能保留 |
| **LibreOffice** | 10k+ | 命令行批量转格式：Word↔PDF↔ODT↔HTML |
| **Pandoc** | 40k+ | 万能文档格式转换器：Markdown↔Word↔PDF↔HTML↔LaTeX↔EPUB |
| **Draw.io** | 50k+ | 流程图/架构图/思维导图，拖拽式操作 |
| **Slidev** | 40k+ | 用 Markdown 写 PPT，代码块直接变幻灯片 |
| **PPTist** | 8k+ | 网页版 PPT 编辑器，跟 PowerPoint 一样用 |
| **Reveal.js** | 70k+ | HTML 演示框架，做炫酷网页 PPT |
| **Excalidraw** | 100k+ | 手绘风格白板，画草图/流程图/架构图 |

> 看到哪个感兴趣，直接把名字告诉韭菜盒子：「帮我装 Stirling PDF」。

## 三、PPT 设计制作

### 方式一：用 AI 直接生成

对韭菜盒子说：

```
帮我做一份 PPT，主题是「2025 电商趋势分析」，
10 页左右，蓝白商务风格
```

AI 会：
1. 自动搭大纲（封面→目录→每页标题→总结→谢谢页）
2. 生成每页内容（标题+要点+数据占位）
3. 输出 `.pptx` 文件，可以下载直接用

### 方式二：Markdown 写 PPT（Slidev）

更灵活——用 Markdown 写内容，AI 帮你排版：

```
帮我用 Slidev 做一份技术分享 PPT，主题「AI 如何改变电商」
```

好处：Markdown 改内容超快，代码块直接渲染，主题一键切换。

### 方式三：手绘风 PPT（Excalidraw）

```
帮我用 Excalidraw 画一个项目架构图
```

手绘风格，适合演示、教学、方案讨论。

## 五、不用装软件

以上所有操作，全在韭菜盒子里完成。你不需要：

- ❌ 装 Python、Node.js、Docker
- ❌ 配环境变量、pip install、npm install
- ❌ 学命令行操作
- ❌ 买 Office 365 订阅（格式转换用开源工具，免费）
- ❌ 装 Adobe Acrobat（PDF 操作用 Stirling PDF，免费）

你只需要对韭菜盒子说一句话。AI 在后台把开源工具装好、跑完、返回结果。

## 常见问题

### 开源项目安全吗？
推荐的都是 GitHub 上高 Star 的知名项目，开源代码透明，不涉及隐私泄露。

### 处理速度怎么样？
取决于文件大小和项目复杂度。一般 PDF 操作几秒，大文件转格式几十秒。

### PPT 能导出 PowerPoint 格式吗？
能。AI 生成的是标准 `.pptx` 格式，Office/WPS/Keynote 都能打开。
