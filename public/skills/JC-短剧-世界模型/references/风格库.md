# 调性分析框架（Style Analysis Framework）

> **核心哲学**：LLM 训练数据覆盖人类所有影视/动漫作品。本文件不提供封闭作品列表——它是一套**调性分析协议**，教模型如何当场拆解**用户提到的任何作品**的调性 DNA。

---

## 一、为什么不用作品列表

- LLM 天然知道王家卫的光影、周星驰的节奏、诺兰的结构、昆汀的暴力美学、新海诚的逆光、今敏的梦境转场……不需要我们列。
- 列 12 个条目 = 限制用户选择范围 = 浪费模型知识。
- 本文件的价值在**分析方法**，不在**答案库**。

## 二、五维调性拆解协议

对用户提到的任何参考作品，当场拆解为 5 个维度。这是后续一切（催化剂/反转/台词/钩子）的驱动源。

### 维度 1：节奏 DNA

| 问自己 | 产出 |
|--------|------|
| 这部作品的情绪推进是快还是慢？ | 节奏类型：快节奏反转 / 中等推进 / 慢炖蓄力 |
| 笑点/爽点/爆点的密度？ | 密度：每 X 分钟一个情绪高峰 |
| 铺垫和爆发的比例？ | 铺垫:爆发 ≈ X:Y |
| 短剧化适配（关键）| 2 分钟一集的短剧需要加速多少？压缩哪些？ |

### 维度 2：台词 DNA

| 问自己 | 产出 |
|--------|------|
| 角色怎么说话？ | 风格标签：嘴炮/冷峻/文白相间/金句频出/话痨/惜字如金 |
| 台词的典型长度？ | ≤10字 / ≤20字 / 长短交替 |
| 有无标志性句式？ | 如「I am Iron Man」式宣告、「曾经有一份真诚的爱情」式独白 |
| 幽默感的来源？ | 冷幽默/屎尿屁/谐音梗/反差萌/讽刺 |

### 维度 3：反转 DNA

| 问自己 | 产出 |
|--------|------|
| 反转的类型偏好？ | 身份反转 / 认知反转 / 装备反转 / 层级反转 / 规则反转 / 宿命反转 |
| 反转的频率？ | 每场 / 每 2-3 场 / 每集末尾 |
| 反转的规模？ | 小反转（台词层面）/ 中反转（关系层面）/ 大反转（世界观层面）|

### 维度 4：情绪 DNA

| 问自己 | 产出 |
|--------|------|
| 基调情绪是什么？ | 燃/悲/爽/笑/紧张/压抑/荒诞/温暖 |
| 是否有情绪反差？ | 单一情绪 / 笑中带泪 / 燃中带悲 |
| 情绪强度？ | 克制收敛 / 中等 / 极致炸裂 |

### 维度 5：冲突 DNA

| 问自己 | 产出 |
|--------|------|
| 冲突的主要来源？ | 人与人 / 人与体制 / 人与自身 / 人与环境 / 人与宿命 |
| 冲突升级方式？ | 蝴蝶效应叠加 / 层层递进 / 突然爆发 / 棋盘式博弈 |
| 反派设计？ | 单一强大反派 / 组织体系 / 自己就是反派 / 没有反派（环境/规则即反派）|

## 三、校准示例（不是菜单，是格式锚点）

以下 3 个示例展示**拆解协议的实际产出**。它们不是"可选菜单"——用户提到其他作品时，当场产出相同格式。

### 示例 A：昆汀《杀死比尔》

```
## 调性 DNA — 昆汀《杀死比尔》

- 节奏：章节式推进，每段战斗风格不同。铺垫:爆发 = 4:6，每章一次高潮战斗。
  短剧适配：每集 = 一章的浓缩，开场直接进战斗或复仇对话。
- 台词：冷峻、简洁、名台词反复出现。≤12字为主，关键句重复变奏。
- 反转：层级反转——以为杀完了→幕后还有→幕后的幕后。每章末一个中反转。
- 情绪：冷酷 + 愤怒 + 复仇快感。单一高强度，不搞笑。
- 冲突：复仇者执念 vs 自己的道德底线。一对一逐步升级。

### 催化剂设计原则
优先：复仇路上撞见更大的仇人、看似无关的人其实是关键、以为的终点是新的起点
避免：日常误会、搞笑桥段、温馨场景

### 标志性元素可借用
章节标题卡、黑白/彩色切换、战斗前对视静默、血不喷而涌
```

### 示例 B：周星驰《功夫》

```
## 调性 DNA — 周星驰《功夫》

- 节奏：快节奏反转，笑点密集。铺垫:爆发 = 2:8，几乎不铺垫。
  短剧适配：天生适合短剧——已有的 3 分钟节奏直接可用。
- 台词：夸张、接地气、金句频出。≤15字，市井味，脏话带哲理。
- 反转：身份反转是核心——小人物其实是绝世高手。物品反转：垃圾=宝贝。
- 情绪：笑中带泪，荒诞中见真情。前半场笑→后半场泪→结尾温暖。
- 冲突：底层互害→发现真正的敌人→共同对抗大反派。

### 催化剂设计原则
优先：荒诞误会、身份错位、阴差阳错、小人物的倒霉运
避免：正经战斗、严肃对话、长篇说教

### 标志性元素可借用
猪笼城寨式群像、斧头帮式反派排场、如来神掌式绝招名字、酱爆式配角抢戏
```

### 示例 C：新海诚《你的名字。》

```
## 调性 DNA — 新海诚《你的名字。》

- 节奏：抒情慢推→中段加速→结尾冲刺。铺垫:爆发 = 5:5。
  短剧适配：每集 2 分钟需压缩日常片段，保留情感爆发点和关键反转。
- 台词：日常感 + 偶尔深情独白。对话自然不做作，独白充满画面感。
- 反转：身份反转（身体互换）→ 时空反转（跨越时间）→ 宿命反转（彗星）。
  层层嵌套。
- 情绪：温暖 + 遗憾 + 最后释然。情绪曲线从甜到虐再到治愈。
- 冲突：时空阻隔 + 记忆消失 vs 想要重逢的执念。

### 催化剂设计原则
优先：发现对方的痕迹、时间线错位的证据、即将错过前的最后努力
避免：外部反派、暴力冲突、阴谋

### 标志性元素可借用
逆光大景、手机/日记作为叙事工具、奔跑（情感爆发标配）、
「你的名字是？」式关键句、天气/光线变化作为情绪标记
```

## 四、杂交融合协议

用户选 A+B 杂交时，不是简单相加——要找**化学反应点**。

### 杂交三步法

1. **找交集**：A 和 B 有什么共同 DNA？例：周星驰的「身份反转」+ 钢铁侠的「装备反转」→ 交集 = 反转爽感。杂交时强化这个交集维度。

2. **互补**：A 有什么 B 没有？例：周星驰的「市井喜剧」补钢铁侠的「高冷科技」→ 「市井科技喜剧」。催化剂「法器当废铁卖→反而淬炼升级」就是互补产物。

3. **防冲突**：检查有没有互斥维度。例：韩剧复仇的「克制冷静」 + 周星驰的「夸张搞笑」→ 可能打架。需明确**主调性**和**辅调性**：谁是底色，谁是调味。

### 杂交输出格式

```
## 杂交调性 DNA — A《X》× B《Y》

- 主调性：[A 或 B]，理由：[为什么选它做主]
- 辅调性：[另一个]，注入：[具体注入什么元素]
- 化学反应点：[A 和 B 碰撞产生的独特东西]
- 节奏：[融合后的节奏]
- 台词：[融合后的台词风格]
- 反转：[融合后的反转偏好]
- 情绪：[融合后的情绪基调]
- 冲突：[融合后的冲突偏好]

### 催化剂设计原则
[融合后的催化剂偏好，标注哪些来自 A、哪些来自 B]
```

## 五、短剧适配调整

电影/长篇动画的节奏不能直接套到 2 分钟短剧。分析完参考作品后必须做适配：

| 维度 | 电影/长篇 | 短剧适配 |
|------|----------|----------|
| 铺垫深度 | 可以有 10-20 分钟铺垫 | 压缩为 1-2 句台词或一个镜头 |
| 情绪曲线 | 缓慢爬升→高潮→回落 | 开场即爬升，全程不回落 |
| 钩子密度 | 每 20-30 分钟一个 | 每集末尾必有钩子 |
| 群像深度 | 可以慢慢展开配角 | 配角扁平化，标签化，不超过 2 个特征 |
| 反转规模 | 可以慢慢铺大反转 | 小反转每场，中反转每集，大反转每 5-10 集 |

## 六、反模式（不要做的事）

| ❌ | 为什么错 |
|----|----------|
| 「电影感」「大片质感」等抽象词 | 无法执行。必须具体到某部作品的某个特征 |
| 同时推荐同一个导演的 3 部作品 | 不是真正的差异化选择 |
| 杂交时两个调性五五开 | 等于没有调性。必须明确主辅 |
| 忽略作品类型与短剧题材的兼容性 | 文艺片节奏不适合爽剧，除非刻意反差 |
| 只复制视觉风格忽略了叙事节奏 | 调性不只是画面，核心是节奏 |
| 推荐了作品但说不出「为什么这部适合」 | 模型必须结合建制设定给出适配理由 |

---

## 七、模型自主推荐策略

调性阶段推荐 A/B/C 时，模型应：

1. **读建制设定**（世界观/主角性格/爽点偏好）→ 理解故事底色
2. **从训练数据中搜索** → 找 3 部调性差异明显的作品（不限库内）
3. **当场拆解** → 每部作品出五维 DNA（用 §二 协议）
4. **标注适配理由** → 为什么这部作品适合这部短剧

**推荐覆盖原则**：3 个选项应朝 3 个方向拉开——如一个幽默向、一个燃向、一个暗黑向。让用户有明显取舍。

**用户说出不在任何库里的作品时**：不要慌。用 §二 协议当场拆解。模型天生会这个。
