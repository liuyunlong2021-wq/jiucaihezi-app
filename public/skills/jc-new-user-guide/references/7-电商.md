# 电商：用 AI 做产品图、视频、数字人带货

> 不用请摄影师、不用找模特、不用租影棚。韭菜盒子创作面板 + 画布一条龙，咔咔出商用素材。

---

## 一、提示词参考站：dazi.studio

写不好 AI 提示词？直接去 **[dazi.studio](https://dazi.studio/)** 抄。

这网站是专门给电商和自媒体准备的 AI 提示词参考站，分类收藏了大量高质量提示词模板，覆盖：

- 产品摄影（白底图、场景图、模特手持）
- 美妆护肤
- 服装鞋包
- 食品饮料
- 家居数码
- 视频广告

找到你想要的风格 → 复制提示词 → 粘贴到韭菜盒子创作面板 → 换成你的产品 → 出图。

---

## 二、创作面板：生图 + 生视频

打开韭菜盒子 → 点「创作」标签。

### 店铺主图 / 首图

选图片模型（GPT Image 2 或 Midjourney.1），贴提示词：

```
电商产品摄影，一瓶精华液放在极简白色台面上，
柔光从左侧打过来，玻璃瓶身有光斑折射，
背景干净虚化，构图居中，留白给文字，
高清，商业摄影质感
```

出图后不满意？点进**画布**微调。

### 产品细节图

同一个产品，换角度、换场景：

```
特写镜头，精华液滴管从瓶中提起，
一滴精华液悬挂在滴管尖端，
微距，高速摄影质感，黑色背景
```

### 产品宣传视频

切到视频模型（Seedance 2.0），把上面图片做首帧，加运镜提示词：

```
相机从产品正前方缓慢推进到特写，
柔光流动，产品表面有高光扫过，
最后定格在品牌logo位置
```

## 三、画布：精细改图

创作面板出的图，点「在画布中编辑」进去。

画布里能干啥：

| 功能 | 用途 |
|------|------|
| **局部重绘** | 圈一块 → 告诉 AI 怎么改（比如把白色背景改成木纹桌面） |
| **扩图** | 图片不够大，AI 自动向外扩展画面 |
| **抠图去背景** | 一键抠出产品，换纯白/透明背景 |
| **文字/Logo** | 在图上叠字、加品牌标 |
| **调色** | 亮度、对比度、饱和度，跟修图软件一样 |

不用学 Photoshop，画布里全搞定。

## 四、反推提示词

### 反推照片提示词

你看到一张竞品图，想知道它用的什么提示词——把图上传到韭菜盒子，点「反推提示词」，AI 自动分析：

- 构图方式
- 光线方向
- 道具搭配
- 调色风格
- 景深

然后你拿着这个提示词，换成你自己的产品，直接出同款质感的图。

### 反推视频提示词

同理，上传一段产品视频，AI 分析运镜、转场、节奏，吐出视频提示词，你在 Seedance 2.0 里复刻。

## 五、数字人带货

韭菜盒子有数字人功能，分两档：

| 类型 | 怎么用 | 适合 |
|------|--------|------|
| **极速数字人** | 上传一张照片 + 一段音频，数字人开口说话 | 快速试品、批量产出 |
| **数字人** | 首帧图 + 台词 + 参考音频，精细控制表情口型 | 主推款、品牌视频 |

场景：产品介绍口播、使用教程、买家秀替代——不用真人出镜。

---

## 六、口播视频

产品介绍文案写好了？走口播管线：

`帮我做一条口播视频：[你的文案]`

AI 自动：转写 → 生成花字字幕（产品名高亮、价格强调、卖点卡片）→ FFmpeg 合成竖屏视频。

一条龙，直接发抖音/快手/小红书。

## 常见问题

### 出图能不能商用？
能。韭菜盒子生成的图片版权归你，直接上架商品不侵权。

### 会不会跟别人撞图？
不会，因为你的产品、你的提示词、你的画布微调是独一无二的。用 dazi.studio 参考的是风格方向，不是照搬。

### 不会写提示词怎么办？
dazi.studio 直接抄，或者把竞品图丢进「反推提示词」。
