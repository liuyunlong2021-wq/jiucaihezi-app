# 字幕校准规则

- `subtitle_raw.srt` 是 Whisper 原始结果，只读保留。
- 输出 `subtitle_corrected.srt` 前，完整通读上下文，不逐条孤立猜词。
- 东北话必须保留原味；不把“嘎哈、咋整、贼、老鼻子、得劲”等改成书面普通话。
- AI 术语统一为 `AI`、`ChatGPT`、`Claude`、`Codex`、`MCP`、`提示词`、`Agent`、`API`。
- 数字、比例、金额、型号、人名和品牌不确定时进入 `subtitle_quality_report.json`，不能静默猜测。
- 重复口吃可清理，但不能删除改变语气或喜剧效果的东北话重复。
- 先按字幕时间轴裁剪超过 1.5 秒的无对白停顿，之后所有字幕与高光使用重映射时间。
- 常驻字幕文本和高光强化文本分离，不能用强化文案替换真实对白。
