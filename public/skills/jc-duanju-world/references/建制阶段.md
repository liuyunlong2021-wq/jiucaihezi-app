# 建制阶段：逐轮收集灵感

**格式铁律**：每个问题提供 3 个具体选项（A/B/C）+ 用户自定义（D/E/F）。

---

## 第 0 轮：初始探索

先问一句，判断用户想法的完整度：

```
你想写一个什么样的短剧？哪怕只是一个模糊的想法也可以——
比如"修仙世界复仇"、"现代都市逆袭"、"悬疑惊悚"等等。
```

**完整性判断**：
- ✅ 用户已说清世界观 + 主角 + 困境 → 跳过收集，直接进入第 5 轮（关键设定）
- ⚠️ 用户只说了模糊方向 → 从第 1 轮开始逐轮收集

---

## 第 1 轮：世界观 & 基调

```
先确定两个最基础的东西——

【故事基调】

请选择（例如：C;A）：
```

## 第 4 轮：对立面 & 世界规则

```

A. 强大组织/宗门（主角要对抗一个庞然大物）
B. 个人宿敌（一个具体的死对头）

【世界特殊规则】—— 这个世界有什么与众不同的规则？

```

A. 完全可用（开局就强，一路碾压）

【爽点偏好】—— 你希望故事的爽感主要来自哪里？

## 建制完成 → 摘要确认

```
## 建制摘要

- 世界观：修仙世界 · 热血复仇
- 主角：林风 | 内核=复仇 | 性格=隐忍、狠辣、重情义
- 身份：被灭门遗孤 · 乞丐 | 困境：被追杀
- 时间：古代 | 反派：强大宗门 | 规则：修仙等级
- 能力：可用但削弱 | 共情：不公平伤害 | 爽点：扮猪吃虎

确认无误？输入 1 继续，或指出要修改的部分。
```

**确认后 → 立即进入调性阶段**（`references/调性阶段.md`）：
1. 媒介分类（真人/动漫）
2. 推荐 3 个具体风格 + 自定义 + 杂交融合（A/B/C/D/E）
3. 锁定调性 → 写入 `wiki/世界/调性.md`

调性锁是一切后续的基础——催化剂、反转、台词、冲突升级都以此驱动。

---

## 第 1 场：先选催化剂（调性驱动），再生成

**第 1 场也需要选择催化剂**。催化剂 = 故事起点如何呈现。
**催化剂选项必须带调性风味**——读取 `wiki/世界/调性.md`，根据锁定的调性生成选项。

```
# 第 1 场催化剂 — 如何切入？（调性：[简述当前调性]）

### 选项A：[带调性风味的切入方式]
从 [某个冲突瞬间] 直接切入，[融入调性特征]

### 选项B：[带调性风味的切入方式]
从 [某个日常场景] 切入，用平静衬托即将到来的风暴，[融入调性特征]

### 选项C：[带调性风味的切入方式]
从 [某个关键事件] 切入，倒叙/闪回到最震撼的一刻，[融入调性特征]

### 选项D：用户自定义
请输入您希望的切入方式：

请选择（A/B/C/D）：
```

**示例**（调性 = 周星驰无厘头 × 漫威钢铁侠）：
```
### 选项A：从天而降的锅
林风刚修好五爷的悬浮飞板，一锅热汤从三楼泼下来——五爷的新女友把他当成了小三。林风（擦脸）："这锅……是珐琅的？挺值钱啊。"

### 选项B：欠租倒计时
贫民窟楼道贴着「欠租 72 小时强制纳米清退」。林风盯着自己只剩 4% 电量的机械臂——今天再不交租，门锁会自动把他传送到城外的辐射区。

### 选项C：闪回重启
△ 系统重启界面闪过血红色字符「复仇程序 v3.0 加载中」。闪回：三年前，林氏科技大厦爆炸。镜头回到现在——林风猛地睁眼，机械臂嗡鸣。
```

**第 1 场特殊性**：必须建立世界观、展示主角核心特质、制造悬念。

**开场结构**：冲突展示（前 3 秒）→ 身份暗示 → 行动选择 → 悬念设置。

用户选择后，生成第 1 场：

```
## 场1-1 已生成，请确认

[展示完整场景内容]

1. 满意 - 继续  2. 不满意 - 重新生成  3. 需要修改

请选择（1/2/3）：
```
