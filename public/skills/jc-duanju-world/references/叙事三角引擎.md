# 叙事三角引擎 —— 角色 × 道具 × 场景 三体联动

> **核心升级**：催化剂不只触碰角色红线。道具的物理法则和场景的空间逻辑，本身就是冲突生成器。LLM 在设计催化剂时多想两个维度——道具能怎么参与？场景能怎么放大？——戏剧张力自然翻倍。

## 一、三角咬合原理

```
           角色引擎（谁在行动 / 红线 / 决策算法）
           /            \
          /   催化剂 ×    \
         /                \
   道具引擎             场景引擎
（物理法则·代价）     （空间逻辑·情绪）
```

一个真正的三角催化剂 = **角色的红线被道具的物理法则逼到场景的情绪锚点里**。

### 道具如何驱动叙事

| 叙事功能 | 说明 | 示例 |
|---------|------|------|
| 麦高芬 | 所有人都想要的東西 → 驱动所有角色行动 | 魔戒、圣杯 |
| 资格认证 | 只有特定条件才能使用 → 身份揭穿/成长证明 | 石中剑 |
| 道德试炼 | 使用有代价 → 每次使用都是人性考验 | 死亡笔记 |
| 时间炸弹 | 倒计时/有限次数 → 制造紧迫感 | 月光宝盒 |
| 真相容器 | 包含关键信息但有获取条件 | 父亲的加密手机 |
| 关系信物 | 绑定关系 → 归还/破坏=关系转折 | 婚戒、徽章 |

### 场景如何驱动叙事

| 叙事功能 | 说明 | 示例 |
|---------|------|------|
| 权力压缩 | 空间结构天然压制一方 | 阿房宫式：走廊走三分钟气势减半 |
| 身份外化 | 空间=角色内心的物理表达 | 蝙蝠洞：孤绝的庇护所 |
| 规则密室 | 进入=被迫参与游戏 | 对角巷式：隐藏世界的入口 |
| 情绪陷阱 | 空间制造情绪，角色被迫暴露 | 酒池肉林：享乐后的虚无 |
| 时间胶囊 | 空间封存了过去 | 父亲书房：一切保持原样 |

### 三角咬合示例

```
角色：哈姆雷特式总裁（思辨拖延——越想冷静越紧张）
道具：石中剑式印章（只有真正冷静才能激活的生物锁）
场景：阿房宫式办公室（走廊走三分钟 / 无窗 / 无椅）

→ 三角催化剂：
仇人坐在办公桌后，把印章推过来。
「签了转让协议。哦——得用你爸的印章。」
三分钟前，主角走那条走廊时还在做心理建设。
现在他拿起印章——手在抖，不亮。每个人都在看。
```

三股力同时作用：哈姆雷特引擎（越紧张越不冷静）→ 石中剑引擎（不冷静=不激活→当众羞辱）→ 阿房宫引擎（走走廊时气势已泄 + 无椅=无处可退）。

## 二、催化剂设计升级

### LLM 内部思考流程（不输出给用户）

设计催化剂时，LLM 在心里问自己三个问题：

```
1. 角色的红线是什么？催化事件如何触碰这条线？
2. wiki/道具/ 里有可用的道具吗？它的物理法则能否嵌入催化事件？
3. 当前场景的空间逻辑是什么？能否放大催化事件的压迫感？

→ 能咬合一个维度 = 合格催化剂
→ 能咬合两个维度 = 强催化剂
→ 能咬合三个维度 = 三角催化剂（优先推荐为选项A）
```

### 催化剂选项模板

```
# 催化剂选项 — 场X-Y

主角：[名字]，压力值：X/100
当前场景：[场景名]（[空间简述]）
可用道具：[道具名]（[简述]）—— 没有就省略此行

### 选项A：[名称]  ★ 推荐
[催化剂描述——优先设计三角引擎]
⏱ [本选项的时间压力——倒计时/窗口期/时机——没有就省略]
📍 [本选项的空间维度——距离/位置差/在场/缺席——没有就省略]

### 选项B：[名称]
[催化剂描述]

### 选项C：[名称]
[催化剂描述]

### 选项D：自定义
```

**关键**：引擎级别（三角/双引擎/单引擎）是 LLM 生成选项时的**内部质量检查**，不输出给用户。用户只需要看「哪个选项的戏剧效果最对味」。

## 三、麦高芬

麦高芬 = 叙事功能为「终极目标」的道具——不是谁用它，而是所有人想要它。

当 `wiki/道具/` 中存在满足以下条件的道具时，LLM 自然提高其优先级：
- 多方想要、改变规则、可转移、有时间窗口

不需要额外追踪系统。LLM 每场重读 wiki/道具/，自然知道哪些道具驱动剧情。

## 四、核心原则

1. **不建追踪系统**：不用手工记录「激活状态」「未用元素」。LLM 每场重读 wiki，直觉判断什么可用。
2. **不标引擎级别**：三角/双引擎/单引擎是 LLM 的内部思考，不输出给用户。
3. **道具场景用到了才激发**：不在建制阶段全做完，用到时自然想起原型激发。
4. **宁少勿多**：没有可用道具就老老实实做单引擎催化剂。强行编一个「三角引擎」比不做更差。
