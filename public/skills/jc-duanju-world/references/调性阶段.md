# 调性阶段：锁定风格调性

> **触发时机**：建制摘要确认后，第 1 场生成前。
> **核心目的**：通过用户确认的媒介 + 导演作品风格，立刻锁定这部剧的调性，后续所有催化事件、反转、台词、节奏都按此调性生成。
> **参考来源**：`references/风格库.md`

---

## 步骤 5.5a：媒介分类

建制摘要确认后，先确认媒介：

```
# 确定媒介类型

在开始写之前，先确认一件事——

这部剧你想做成什么样的视觉呈现？

**A. 真人剧** — 真人演员出演，对标真人影视作品
**B. 动漫** — 动画/漫画风格，对标动漫作品

请选择（A/B）：
```

记录到状态中（存入 `wiki/世界/调性.md`）。

---

## 步骤 5.5b：推荐风格（A/B/C + D 自定义 + E 杂交）

根据已确认的**建制设定**（世界观/基调/主角/爽点）和**媒介类型**，模型从自身训练数据中搜索，推荐 **3 个具体作品风格**，覆盖不同方向。

> **模型不限于任何预设库**。LLM 训练数据覆盖了人类所有影视/动漫作品——用 `references/风格库.md` 的「五维拆解协议」当场分析即可。

每个推荐必须包含：
- `option_id`: A / B / C / D / E
- `source`: 具体作品，如「昆汀《杀死比尔》」
- `why`: 结合建制设定，一句话说明为什么这部作品适合
- `tone_design`: 用五维协议当场拆解（节奏 + 台词 + 反转 + 情绪 + 冲突）

**D 选项** = 「自定义：告诉我想参考的导演/作品，任何作品都可以」
**E 选项** = 「杂交融合：选两个以上风格混合，如 A+B 或 你自己说两个作品」

**判定标准**：3 个选项应覆盖不同方向（如一个幽默向、一个燃向、一个暗黑向），让用户有明显取舍。

**不允许**：
- 只给抽象词（如「喜剧风」「爽剧」）
- 给不出具体作品
- 推荐同导演的不同作品当两个选项
- 只从 `references/风格库.md` 的校准示例里选——那是格式锚点，不是菜单

### 交互模板

```
# 锁定调性 — 这部剧的「味道」

基于你的设定（[简述建制摘要]），我推荐 3 种风格方向：

### 选项A：[作品名]（[导演/公司]）
**风格标签**：[2-3 个关键词]
**为什么适合**：[结合你的世界观/主角/爽点的理由]
**调性预览**：
- 节奏：[简述]
- 台词风格：[简述]
- 反转偏好：[简述]
- 情绪基调：[简述]

### 选项B：[作品名]（[导演/公司]）
**风格标签**：[2-3 个关键词]
**为什么适合**：[理由]
**调性预览**：
- 节奏：[简述]
- 台词风格：[简述]
- 反转偏好：[简述]
- 情绪基调：[简述]

### 选项C：[作品名]（[导演/公司]）
**风格标签**：[2-3 个关键词]
**为什么适合**：[理由]
**调性预览**：
- 节奏：[简述]
- 台词风格：[简述]
- 反转偏好：[简述]
- 情绪基调：[简述]

### 选项D：自定义
请输入你想参考的导演/作品名，或描述你想要的风格。

### 选项E：杂交融合
选 2 个以上风格组合，例如「A+B」、「A+C」或「A+B+你自己的配方」。
杂交示例：周星驰无厘头 × 漫威钢铁侠 = 嘴炮超英+科技道具搞笑+打斗中插段子

请选择（A/B/C/D/E）：
```

## 步骤 5.5c：锁定调性

用户选择后：

1. 若 A/B/C：用**五维拆解协议**（`references/风格库.md` §二）当场拆解 → 写入 `wiki/世界/调性.md`
2. 若 D：用户可能说出任何作品。模型用五维协议当场拆解。如果用户给的是抽象描述（如"王家卫那种感觉"），追问到具体作品才继续。
3. 若 E：用**杂交三步法**（`references/风格库.md` §四）生成融合调性，向用户确认后写入

### 写入 `wiki/世界/调性.md`

```markdown
---
title: 调性
tags: [短剧, 调性]
---

## 媒介
[真人 / 动漫]

## 参考作品
- 主参考：[导演/公司]《[作品名]》
- 杂交元素（如有）：[导演/公司]《[作品名]》的 [具体元素]

## 调性 DNA（五维）

### 节奏
[节奏类型 + 密度 + 铺垫:爆发比 + 短剧适配说明]

### 台词风格
[风格标签 + 典型长度 + 标志性句式 + 幽默来源]

### 反转 DNA
[反转类型偏好 + 频率 + 规模]

### 情绪基调
[基调情绪 + 情绪反差 + 强度]

### 冲突偏好
[冲突来源 + 升级方式 + 反派设计]

### 催化剂设计原则
[基于以上五维，催化剂优先什么、避免什么]

### 标志性元素可借用
[该作品在叙事层面可借用的标志性手法，不是视觉风格]
```

## 调性锁定后的连锁效应

一旦调性锁定，后续所有环节必须据此调整：

| 环节 | 调性驱动方式 |
|------|-------------|
| 催化剂选项 | 选项中融入调性风味。如无厘头风 → 催化剂必带荒诞/错位元素 |
| 反转设计 | 反转模式必须匹配调性。如超英风 → 「绝境→觉醒→华丽反杀」 |
| 台词 | 台词语气、长度、金句密度匹配调性 |
| 冲突升级 | 冲突升级方式匹配调性。如黑色喜剧 → 蝴蝶效应叠加 |
| 钩子 | 钩子类型偏好匹配调性。如悬疑风 → 悬念钩子优先 |

详细规则见 `references/写作质量手册.md` §十六「调性驱动写作」。
