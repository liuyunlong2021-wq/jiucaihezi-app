# 写作工作流（阶段8-9）

## 阶段 8：正文写作

### 逐章写作循环

```
第1章特殊：开篇三选一
  A. 冲突切入（战斗中/危机中）
  B. 反常设定（独特世界观展示）
  C. 悬念倒叙（精彩片段前置）

S2起逐章循环：
  1. 读 CLAUDE.md → 进度/压力值/待收伏笔
  2. 催化剂选择（A/B/C/D）
  3. 生成章节 2000-4000 字
  4. 用户确认
  5. 更新角色/悬念/时间线/CLAUDE.md
```

### 章节质量要求

对照 `references/写作质量手册.md`：
- 中心事件明确
- 对话有功能
- 冲突 3-5 层
- 章末有钩子
- 压力值驱动下一章

### 分批交付
- 每 5-10 章一批，用户确认后继续
- 每批结束后建议调用 JC-jiyiyasuo 压缩记忆

**产出**：`wiki/剧本/第X章.md`（逐章追加）

## 阶段 9：复盘分析

**目标**：全稿诊断报告。

检查项：
- [ ] 主线是否清晰贯穿
- [ ] 角色弧光是否完整
- [ ] 伏笔是否全部回收
- [ ] 节奏是否均匀
- [ ] 字数是否达标

**产出**：复盘报告
