# SKILL-04: 情感数值化引擎

## 核心职责

用数值监控感情线，避免忽快忽慢或直线发展

## 坐标定义

```
-10 ■■■■■■■■■■ 杀父之仇级别的恨
-5  ■■■■■         厌恶、排斥
0   ○○○○○         陌生路人
+3  ■■■           有好感
+5  ■■■■■         喜欢
+7  ■■■■■■■       爱
+10 ■■■■■■■■■■ 生死相许
```

## 大纲打点表格

| 章节 | 分值 | 变化 | 触发事件 | 备注 |
|------|------|------|---------|------|
| 1-3 | 0 | — | 初遇 | 陌生 |
| 4-10 | +2 | +2 | 特殊对待 | 好奇 |
| 11-20 | +4 | +2 | 守护事件 | 好感 |
| 21 | +1 | -3 | 误会爆发 | 🔻减分！|
| 30-40 | +6 | +5 | 误会解开 | 🔺大反弹！|
| 50-80 | +7 | +1 | 日常相处 | 升温 |
| 100 | +8 | +1 | 表白 | 质变 |
| 150 | +9 | +1 | 确立关系 | 稳定 |
| 280 | +10 | +1 | 生死考验 | 终极 |

## 推拉理论

### 核心原则

```
✅ 必须有"推（加分）"和"拉（减分）"的波动
❌ 不能是0→3→5→7→10的直线
✅ 每次减分后的反弹必须比之前高
```

### 波浪曲线

```
分值
+10 |                          *
+8  |                   *
+6  |            *
+4  |      *
+2  |  *
0   *
-3  |     *（误会）
    |___________________________章节
```

## 触发事件设计

### 加分事件（+1到+3）

- 守护、关心、吃醋
- 肢体接触、暧昧互动
- 特殊对待、双标

### 减分事件（-1到-5）

- 误会、冲突
- 价值观碰撞
- 第三者插足
- 危机/背叛（假象）

### 质变节点（关键分数）

- 0→+3：从陌生到有感觉
- +5→+7：从喜欢到爱
- +7→+10：从爱到生死相许

## 检查清单

- [ ] 是否画出了情感曲线图？
- [ ] 是否有至少1次减分-反弹？
- [ ] 减分是否有合理动机？
- [ ] 加分是否有足够铺垫？
- [ ] 某章分值不变是否过多？（>5章=无效剧情）
