# SKILL-02: 内线与主题审判器

## 核心职责

为故事设计深层主题（内线），通过层层审判考验主角

## 核心概念

```
外线 = 发生了什么事（剧情）
内线 = 这个故事要说什么（主题/立意）
```

## 审判层级设计

### Step 1: 确定核心主题

常见主题库：

- **独立**：不需要任何人的认可
- **真实**：做自己，不需要伪装
- **勇气**：面对不公要反击
- **救赎**：原谅与释放
- **爱**：被爱与爱人
- **成长**：从幼稚到成熟

### Step 2: 设计审判层级

```
核心主题：独立

第一层审判（社会层面）：
├─ 考验：豪门否定她的价值
├─ 选择：接受评价 VS 活出自己？
└─ 回应：我不需要你们的认可

第二层审判（亲密关系）：
├─ 考验：依赖男主 VS 独立解决？
├─ 选择：示弱求助 VS 自己守护？
└─ 回应：我有能力守护自己

第三层审判（自我层面）：
├─ 考验：豪门跪求原谅，回归家族 VS 独立生活？
├─ 选择：依附 VS 独立？
└─ 回应：我可以原谅，但不会依附

终极审判：
└─ 回应：我有资格站在聚光灯下
```

## 关键原则

1. **一致性**: 内线要贯穿全文，不能中途变化
2. **反派设计**: 反派是违背主题的人
3. **高潮对应**: 每个大情节都在考验主题

## 检查清单

- [ ] 核心主题是否明确？
- [ ] 是否设计了3-4层审判？
- [ ] 每层审判是否有明确的考验、选择、回应？
- [ ] 反派是否违背了主题？
- [ ] 主角的成长是否围绕主题展开？
