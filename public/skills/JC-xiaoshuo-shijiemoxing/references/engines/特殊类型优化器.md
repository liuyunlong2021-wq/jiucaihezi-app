# SKILL-05: 特殊类型优化器 (Special Type Optimization)

> 💫 "针对不同流派的专项优化。感情流要推拉，剧情流要紧凑。"

---

## 核心职责

针对**感情流**和**剧情流**两种不同类型的章节，应用特定的优化规则，确保节奏符合该流派读者的预期。

---

## 感情流专项：三浅一深法则 (Three Shallow One Deep)

**适用**：CP互动、感情推进章节。

**法则内容**：
感情不能一步登天，也不能一潭死水。合理的推进节奏是 **3次浅层互动 + 1次深层互动** 的循环。

### 浅层互动 (Shallow Interaction) - 🟡

- **定义**：日常的、微小的、试探性的接触。
- **示例**：
  - 眼神对视。
  - 肢体无意触碰（指尖碰到）。
  - 简短的关心/互怼。
  - 共同完成一件小事。

### 深层互动 (Deep Interaction) - 🟡🔴

- **定义**：突破性的、决定性的、强烈的情感爆发。
- **示例**：
  - 交心的深夜长谈。
  - 拥抱、牵手、接吻。
  - 生死关头的相救。
  - 重大误会的解除或产生。
  - 明确的表白或拒绝。

**检查指令**：

- 检查最近5章的互动记录。
- 如果连续 >5章 都是浅层 → **警告：进度太慢，读者会急！** → 本章建议安排深层互动。
- 如果连续 >2章 都是深层 → **警告：进度太快，糖发多了会腻！** → 本章转入剧情或浅层。

---

## 剧情流专项："不想写"测试 (The "Boring" Test)

**适用**：所有非高潮章节、过渡章节。

**法则内容**：
有效的情节必须满足 **"塑造人设"** 或 **"推动剧情"** 至少一项。如果一项都不满足，就是无效水文。

**两问法自检**：

1. **Q1：这个情节塑造人设了吗？** (体现了性格、能力、反差？)
2. **Q2：这个情节推动剧情/感情了吗？** (改变了现状、获得了信息、增进了关系？)

**判定**：

- ❌ **全否** → **无效情节，直接删除！** (例如：无意义的吃饭、睡觉、路人闲聊)。
- ✅ **一是一否** → **保留**。
- 🌟 **全是** → **优质情节** (核心戏份)。

---

## 检查清单 (Checklist)

- [ ] **感情流检查**：本章互动是浅层还是深层？是否符合"三浅一深"的大节奏？
- [ ] **无效情节检查**：是否保留了"既不推剧情也不塑人设"的废话？
- [ ] **平衡检查**：感情流章节里是否还有剧情骨架？剧情流章节里是否夹杂了感情糖？
