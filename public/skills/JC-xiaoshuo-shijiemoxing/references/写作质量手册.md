# 小说写作质量手册

## 核心公式

```
每章 = 建立预期 → 推进冲突 → 打破预期（反转）→ 留钩子
```

## 章节节奏

| 段 | 占比 | 内容 |
|----|------|------|
| 起 | 10% | 承接上章钩子，抛出本章催化剂 |
| 承 | 80% | 冲突升级，角色反应，新矛盾产生 |
| 转 | — | 反转/揭露/抉择 |
| 合 | 10% | 留钩子，铺垫下一章 |

## 对话铁律

- 每句对话有目的（塑造/推动/强化/交代/递进）
- 贴合人设：什么人说什么话
- 短句优先，能命令别请求
- 一人连续说超过 3 句 → 插入打断/动作/反击
- 禁止无功能寒暄

## 叙述铁律

- 事件推进：抽取关键点，跳过过渡（不写走路/吃饭/睡觉）
- 情绪具象化：不写"他很生气"，写「他捏碎了茶杯」
- 禁止大段背景介绍 → 用对话/动作自然带出
- 禁止心理活动长篇大论 → 转化为台词/动作/表情

## 钩子法则

| 类型 | 说明 |
|------|------|
| 悬念钩子 | 信息差，读者想知道后续 |
| 断章钩子 | 高潮中途卡断 |
| 出场钩子 | 关键时刻人物/势力登场 |
| 情绪钩子 | 强烈情绪定格 |

## 创作禁忌

- 主线不明
- 节奏慢/铺垫多
- 配角过多
- 逻辑不通
- 反派弱智
- 一帆风顺
- 大段独白/说教
- 伏笔不收

## 每章自检

- [ ] 中心事件明确
- [ ] 开章有冲突
- [ ] 对话/叙述有功能
- [ ] 冲突 3-5 层
- [ ] 章末有钩子
- [ ] 压力值有变化
- [ ] 字数 2000-4000
