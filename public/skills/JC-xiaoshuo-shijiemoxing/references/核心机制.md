# 核心机制

## 极致性系统

| 维度 | 红线 | 催化剂示例 |
|------|------|-----------|
| 自由 | 被束缚/被命令 | 被囚禁 |
| 正义 | 目睹不公/被冤枉 | 无辜者被杀 |
| 复仇 | 仇人逍遥/被羞辱 | 亲人被杀 |
| 爱情 | 失去爱人/被背叛 | 爱人被抓 |
| 权力 | 失去地位/被轻视 | 被罢免 |
| 知识 | 真相被隐瞒 | 发现秘密 |
| 生存 | 死亡威胁 | 绝境逃生 |
| 荣誉 | 名声被毁 | 被公开诽谤 |
| 贪婪 | 财富被夺 | 宝藏被发现 |
| 守护 | 要保护的人受威胁 | 亲人被绑架 |

## 角色档案模板

写入 `wiki/角色/角色名.md`：

```markdown
---
title: 角色名
tags: [小说, 角色]
---
## 角色名
- 极致维度：[维度]（8-10/10）
- 内核动机：[一句话]
- 当前身份：[身份]
- 红线：[不可触碰的底线]
- 压力值：[0-100]
- 性格特点：[3-5个关键词]
- 性格反差：[如外冷内热]
- 行为模式：[在压力下的典型反应]
- 角色弧光：[从XX状态 → 经历XX → 变成XX状态]
- 家庭背景 / 学识 / 经历：[简述]
- 与主要角色关系：[简述]
```

## 压力值系统

| 压力值 | 状态 | 章节功能 |
|--------|------|----------|
| 0-30 | 平静 | 铺垫/日常/喘息 |
| 30-70 | 紧张 | 冲突升级/推进 |
| 70-100 | 爆发 | 高潮/转折/抉择 |

变化规则：触碰红线 +50 / 催化剂触发 +20 / 冲突解决 -20 / 冲突升级 +30 / 两难选择 +15。
每章后重新计算，最高者为下一章主角。

## 催化剂类型

| 类型 | 说明 | 小说示例 |
|------|------|-----------|
| 外部入侵 | 外部力量打破平衡 | 敌人来袭、天灾降临 |
| 内部觉醒 | 角色认知改变 | 发现身世秘密 |
| 关系裂变 | 人际突发变化 | 盟友背叛、爱人离去 |
| 世界突变 | 环境剧烈变化 | 规则改变、时代更迭 |

## 红线 × 压力值 × 催化剂 联动

```
催化事件 → 触碰红线 → 压力值飙升 → 角色被逼到墙角 → 自然反应 → 推动下一章
```

设计催化事件时：先看红线 → 设计事件精确触碰 → 压力值+50 → 角色必须行动 → 产生新矛盾。
