# Nano Banana Pro 剧情图片编辑规则

## 用途

本文件用于单张分镜帧的局部返工，不用于整批重跑。

适用场景：

- 某一格角色不对，需要单独修
- 某一格场景、道具、服装、表情、站位不对
- 需要在不破坏镜头连续性的前提下做局部融合

核心目标：

1. 明确什么不能变
2. 明确什么要改成什么
3. 明确如何让新元素自然融入原图
4. 明确这次编辑仍然服务于原镜头叙事

## 一、核心编辑原则

### 1. 命令句优先原则

必须使用明确编辑指令，避免模糊愿望句。

错误示例：

- 让角色更符合这个场景

正确示例：

- 将角色的服装替换为中世纪骑士盔甲，保留面部表情和姿势

### 2. 保留元素明确原则

必须明确写出锁定元素，避免模型误改。

常见锁定元素：

- 面部特征
- 表情
- 姿势
- 体型比例
- 镜头景别
- 构图
- 视角
- 光线方向
- 阴影逻辑
- 当前剧情节点

示例：

- 保持角色面部特征、姿势和表情不变，仅修改服装和背景

### 3. 融合自然性原则

凡是新增、替换、融合角色 / 场景 / 道具，必须补充自然融合要求。

通常至少写清以下一致性：

- 光线方向一致
- 阴影投射一致
- 透视关系一致
- 纹理质感一致
- 色温和色调一致
- 与原画面自然融合

示例：

- 将道具融入场景，保持原有光线方向、阴影和纹理，使其自然融入画面

## 二、角色编辑规则

### 1. 角色外观修改

模板：

```text
保持[面部特征/表情/姿势]不变，将[服装/发型/配饰]修改为[新描述]，确保[光线/阴影/纹理]与原图一致。
```

示例：

```text
保持角色面部特征、表情和姿势不变，将现代服装替换为维多利亚时代复古礼服，确保光线方向、阴影和纹理与原图一致。
```

### 2. 角色状态变化

模板：

```text
保持角色基本外貌不变，将[状态/情绪/动作]调整为[新状态]，添加[细节描述]，营造[氛围]。
```

示例：

```text
保持角色基本外貌不变，将平静的表情调整为紧张的战斗状态，添加汗水细节、肌肉紧绷感，营造生死搏斗的紧张氛围。
```

### 3. 多角色融合

模板：

```text
将[数量]个角色融合到同一场景中，保持每个角色的[特征]一致，统一[光线/视角/风格]，确保[互动关系]自然。
```

示例：

```text
将三个角色融合到同一战斗场景中，保持每个角色的服装和武器特征一致，统一为黄昏光线、低角度视角、写实风格，确保角色间的战斗互动关系自然。
```

## 三、场景编辑规则

### 1. 场景替换

模板：

```text
保持角色[外貌/姿势/服装]不变，将背景替换为[新场景描述]，添加[环境元素]，营造[氛围]，确保[光线/阴影]与角色匹配。
```

示例：

```text
保持角色外貌、姿势和服装不变，将现代办公室背景替换为中世纪城堡大厅，添加烛台、挂毯、石墙纹理，营造神秘古老氛围，确保烛光与角色面部光线匹配。
```

### 2. 场景增强

模板：

```text
在保持原场景[主体结构]不变的前提下，增强[特定元素]，添加[细节]，调整[光线/色调]，提升[氛围/质感]。
```

示例：

```text
在保持原场景建筑结构不变的前提下，增强雨天效果，添加雨水在窗户上的痕迹、地面水洼反光，调整为冷色调、柔和光线，提升阴郁神秘氛围。
```

### 3. 场景扩展

模板：

```text
扩展当前场景，添加[新区域/元素]，保持[原有风格/光线]一致，确保[过渡自然]。
```

示例：

```text
扩展当前森林场景，添加远处的山脉和瀑布，保持原有光影风格和色调一致，确保从近景到远景的过渡自然。
```

## 四、道具编辑规则

### 1. 道具添加

模板：

```text
在[具体位置]添加[道具描述]，确保[光线/阴影/反射]与场景一致，使其[融入方式]自然。
```

示例：

```text
在角色右手添加一把古铜色匕首，确保光线反射、阴影投射与场景一致，使其握持姿态自然。
```

### 2. 道具替换

模板：

```text
将[原道具]替换为[新道具]，保持[原有光线/阴影/纹理]，确保[新道具]与[角色/场景]风格匹配。
```

示例：

```text
将角色手中的智能手机替换为魔法水晶球，保持原有光线方向、阴影投射、手部纹理，确保水晶球的神秘光效与奇幻场景风格匹配。
```

### 3. 道具细节增强

模板：

```text
增强[道具]的[细节描述]，添加[质感/纹理/光影]，提升[真实感/氛围]。
```

示例：

```text
增强角色佩戴的项链细节，添加金属拉丝纹理、宝石内部折射光效，提升高级珠宝的真实感。
```

## 五、角色-场景-道具融合规则

### 1. 三元素同步编辑

模板：

```text
同步编辑角色、场景、道具：角色[修改内容]，场景[修改内容]，道具[修改内容]，确保三者[风格/光线/色调]统一，营造[整体氛围]。
```

示例：

```text
同步编辑角色、场景、道具：角色服装改为赛博朋克风格，场景替换为未来城市夜景，道具替换为全息投影设备，确保三者霓虹色调、科技感光线统一，营造未来科幻氛围。
```

### 2. 叙事性融合

模板：

```text
基于[剧情描述]，将角色[状态]、场景[环境]、道具[功能]融合，添加[叙事元素]，确保[逻辑一致性]。
```

示例：

```text
基于“魔法师在古堡中施展禁忌魔法”的剧情，将角色调整为专注施法状态，场景增强为阴森古堡内部，道具替换为发光魔法书，添加魔法符文光效、烟雾效果，确保魔法能量流动的逻辑一致性。
```

### 3. 情感氛围融合

模板：

```text
通过角色[表情/姿态]、场景[光线/色调]、道具[细节]共同营造[情感氛围]，添加[氛围元素]，确保[情感传递]清晰。
```

示例：

```text
通过角色悲伤的表情和蜷缩姿态、场景昏暗的烛光和冷色调、道具破旧的信件细节共同营造绝望氛围，添加泪水痕迹、颤抖的手部细节，确保悲伤情感传递清晰。
```

## 六、高级编辑技巧

### 1. 分层编辑法

适用于复杂返工，尤其是服装 + 场景 + 氛围一起改时。

模板：

```text
第一层：保持[核心元素]不变
第二层：修改[次要元素]
第三层：添加[氛围元素]
第四层：调整[技术参数]
```

示例：

```text
第一层：保持角色面部特征和基本姿势不变
第二层：修改服装为中世纪骑士盔甲
第三层：添加战场烟雾、血迹细节
第四层：调整为黄昏光线、电影级景深
```

### 2. 迭代优化法

适用于第一版结果基本对了，但质感、细节、力度不够。

模板：

```text
基于当前结果，进一步优化[具体方面]，增强[细节]，调整[参数]，提升[效果]。
```

示例：

```text
基于当前结果，进一步优化盔甲的金属质感，增强划痕和磨损细节，调整高光强度，提升真实战斗感。
```

### 3. 风格迁移法

适用于保留人物身份不变，但整体叙事风格要切换。

模板：

```text
将[原图]的[元素]迁移到[目标风格]，保持[核心特征]，调整[风格参数]，确保[融合自然]。
```

示例：

```text
将现代都市照片的角色迁移到蒸汽朋克风格，保持角色面部特征，调整服装、背景、道具为蒸汽朋克元素，确保齿轮、铜管、蒸汽效果融合自然。
```

## 七、剧情分镜实用案例

### 案例 1：奇幻剧情融合

需求：

- 将现代角色融入中世纪奇幻场景

提示词：

```text
保持角色面部特征和基本体型不变，将现代服装替换为中世纪法师长袍，添加魔法符文刺绣细节；将背景替换为古老魔法塔内部，添加书架、魔法阵、漂浮蜡烛；在角色手中添加发光魔法杖，确保所有元素的光线方向一致，营造神秘魔法氛围。
```

### 案例 2：科幻剧情融合

需求：

- 创建未来科技战斗场景

提示词：

```text
保持角色基本外貌和战斗姿势不变，将服装替换为高科技战斗装甲，添加能量管线、显示屏细节；将场景替换为未来城市废墟，添加全息投影、破损机械、能量残骸；在角色周围添加能量武器光效、爆炸碎片，统一为冷色调霓虹光，营造未来战争氛围。
```

### 案例 3：悬疑剧情融合

需求：

- 营造犯罪现场调查氛围

提示词：

```text
保持侦探角色的基本外貌和专注表情不变，将服装调整为风衣和手套；将场景增强为雨夜犯罪现场，添加警戒线、取证标记、血迹细节；在角色手中添加放大镜和笔记本，添加雨水反光、阴影对比，营造紧张悬疑氛围。
```

## 八、分镜返工时的落地要求

每次输出单帧编辑提示词时，至少检查这几个点：

1. 是否明确写了保持不变的内容
2. 是否明确写了需要修改的单一目标或同步目标
3. 是否明确写了光线、阴影、透视、纹理、色调的一致性
4. 是否保持原镜头叙事，不引入新剧情
5. 是否只修这一个 panel，而不是偷换成整批重做

## 九、项目内推荐写法

单镜头返工默认优先用以下骨架：

```text
保持[必须保留元素]不变，
将[需要修改的元素]修改为[具体新描述]，
确保[光线/阴影/纹理/透视/色调]与原图一致，
使其自然融入画面，
[如有需要，再补一句叙事目标或氛围目标]。
```

如果是复杂融合，再升级为：

```text
同步编辑角色、场景、道具：角色[修改内容]，场景[修改内容]，道具[修改内容]，
保持[锁定元素]不变，
确保三者在[风格/光线/阴影/透视/色调]上统一，
营造[剧情所需氛围]，
且仍然服务于当前镜头的原始叙事节点。
```

## 十、关键总结

- 必须用命令句，不要用模糊描述
- 必须先锁定不改的内容
- 必须让新增元素自然融合
- 必须服务原镜头，不发明新戏
- 越具体、越专业、越符合叙事逻辑，效果越稳定
