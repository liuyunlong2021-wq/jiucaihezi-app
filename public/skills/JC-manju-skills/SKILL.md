# 短剧生产管线

## 使用方式

**可从任意环节切入。** 说一句就能定位：

| 用户说什么 | 进入哪个环节 |
|-----------|-------------|
| "帮我做一部短剧""剧本到成片" | `JC-manju-shengchan` 全流程一键编排 |
| "先看看什么风格合适" | `JC-manju-fengge` 风格分析 |
| "分析这个剧本的角色" | `JC-manju-juese` 角色分析 |
| "设计场景" | `JC-manju-changjing` 场景分析 |
| "分析道具" | `JC-manju-daoju` 道具分析 |
| "拆解剧本" | `JC-manju-chaijie` 工程手册 |
| "做分镜/镜头设计" | `JC-manju-fenjing` 分镜设计 |
| "设计音色" | `JC-manju-yinse` 音色设计 |
| "配音视频/对口型" | `JC-manju-peiyin` 配音绑定 |
| "合成视频" | `JC-manju-hecheng` 视频合成 |
| "修改某张故事板图" | `banana-storyboard-edit-prompt` 修图 |
| "生成图片/视频/音频" | `JC-meitichuangzuo` 媒体引擎 |

## 完整管线

```
风格分析 → 角色分析 → 场景分析 → 道具分析
                ↓
           工程手册（剧本拆解）
                ↓
           分镜设计 + 九宫格故事板 + 视频 prompt
                ↓
           音色设计 → 配音绑定 → 视频合成
```

- **JC-manju-shengchan**：一键跑全程。
- 所有子 skill 均可独立调用，不需要从头开始。
- `JC-meitichuangzuo` 是底层媒体引擎，负责实际调用 NewAPI 生成图片/视频/音频，其余 skill 发 prompt 给它执行。
