## 📦 安装

打开**韭菜盒子 APP**，在对话框中输入：

```
请帮我把 https://github.com/liuyunlong2021-wq/JC-manju-skills 的 skills 安装到本地 ~/.agents/skills/ 目录
```

AI 会自动 clone 仓库并注册所有 10 个 Skill。

### 手动安装

```bash
git clone https://github.com/liuyunlong2021-wq/JC-manju-skills.git ~/.agents/skills/JC-manju-skills
```

## 🚀 使用

### 一键拍片

```
请用 JC-manju-shengchan 帮我把剧本拍成片：
[粘贴剧本]
```

编排器会按顺序驱动 9 个阶段，每完成一个向你确认。

### 单独使用

也可以单独调用任意 Skill：

```
分析这个剧本的风格选项
设计主角的角色资产
为这场戏设计分镜
```
