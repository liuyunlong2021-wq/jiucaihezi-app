# 改稿终检清单

> 阶段 6（改稿）和阶段 8（终检）的逐条执行手册。逐条打 ✓/✗。

## 一、对话检查（阶段 6-1）

| # | 检查项 | 标准 |
|---|--------|------|
| D1 | 每句对话贴人设？ | 遮住名字，能从语气认出是谁在说话 |
| D2 | 每句对话有目的？ | 揭示信息 / 改变关系 / 制造悬念——三选一 |
| D3 | 无空话水对白？ | 没有"你吃了吗""今天天气真好" |
| D4 | 无讲话稿腔？ | 没有"我认为""我觉得""我们应该" |
| D5 | 短促有力？ | 绝大多数对话 ≤15 字 |
| D6 | 对话推进剧情？ | 拿掉这句对话，剧情会不会停顿？ |

## 二、真实感检查（阶段 6-2）

| # | 检查项 | 标准 |
|---|--------|------|
| R1 | 事理逻辑自洽？ | 前因后果没矛盾 |
| R2 | 常识无误？ | 时间/地点/称谓/流程/职业细节正确 |
| R3 | 细节有功能？ | 每个细节要么塑造人物，要么推动情节，要么调动情绪 |
| R4 | 无空洞堆砌？ | 没有"美丽的""优雅的""令人心碎的"等浮词 |
| R5 | 细节精确？ | "下午三点"比"下午"好；"解放路137号"比"一条街"好 |
| R6 | 共情点到位？ | 读者能在某个细节上「看见自己」 |

## 三、结构改稿（阶段 6-3）

| # | 检查项 | 标准 |
|---|--------|------|
| S1 | 主线单一？ | 一句话能说清整个故事 |
| S2 | 无重复段落？ | 同一信息不会在多个地方反复出现 |
| S3 | 转折有力？ | 每个转折点让读者意外但合理 |
| S4 | 信息密度够？ | 每一段要么推进剧情，要么揭示角色，要么加深悬念 |
| S5 | 伏笔全回收？ | 对照悬念账本，一个不漏 |
| S6 | 无跑题段落？ | 没有与主线无关的支线剧情 |

## 四、终检硬规则（阶段 8）

| # | 规则 | 不通过时回滚到 |
|---|------|---------------|
| 1 | 前 50 字有冲突/悬念/反差 | 阶段 5（重写 S1 开篇） |
| 2 | 主线单一清晰，一句话能说清 | 阶段 2/4（重新提炼 Logline/大纲） |
| 3 | 爆款四要素全部落地 | 阶段 1（重新选题或强化要素） |
| 4 | 第一人称全程一致，无上帝视角 | 阶段 5/6（替换上帝视角段落） |
| 5 | 每节末有钩子 | 阶段 5（补写节末钩子句） |
| 6 | 对话有功能，无空话水对白 | 阶段 6-1（对话打磨） |
| 7 | 真实感自洽，无常识错误 | 阶段 6-2（真实感检查） |
| 8 | 伏笔全部回收 | 阶段 4/5（补收或删伏笔） |
| 9 | 字数在目标范围内 | 阶段 4/5（扩写或精简） |
| 10 | 交付物齐全（正文+标题+简介+标签） | 阶段 7（补投稿包装） |

**未通过项**（如有）：
