# 知乎投稿包装指南

> 定稿后的最后一步：把故事包装成知乎上能点击、能收藏、能传播的投稿包。

## 标题（3 个备选）

标题是知乎第一战场。好标题 = 点击率。

### 标题公式

**公式 1：身份反差 + 钩子**
`「我」是一个[身份]，但[反常事实]。`
例：「我是一个入殓师，昨晚尸体开口说话了。」

**公式 2：时间压迫 + 悬念**
`[时间限制]内，我必须[做什么]，否则[代价]。`
例：「七天之内我找不到凶手，死的就是我。」

**公式 3：结果倒叙 + 信息差**
`[结果]，但没人知道[真相]。`
例：「我杀了她三次，但每次第二天她都完好无损地出现在我门口。」

**公式 4：规则破坏 + 好奇**
`[常识]是错的——[揭示]。`
例：「你从小到大听过的那个都市传说是真的。我就是证据。」

### 标题铁律
- ≤20 字为佳，最多 25 字
- 必须有「我」字（第一人称感）
- 必须有具体信息（不是虚的形容词）
- 三个备选标题用不同公式，覆盖不同读者群

## 简介（≤180 字）

简介 = 标题的放大器。在标题勾住点击后，简介让读者决定「要不要点开全文」。

### 简介结构
```
第 1 句（0-30 字）：承接标题，把钩子放大
第 2 句（30-90 字）：一句话说清主线——谁、什么事、为什么难
第 3 句（90-150 字）：制造两难或悬念——最核心的 tension
第 4 句（150-180 字）：问句收尾——让读者带着问题进入正文
```

### 示例
> 入殓师这个职业，我干了十一年。十一年里见过的尸体比我见过的活人还多。但昨晚那具，是真的不一样——它的眼睛是睁开的。不是死后僵硬的那种睁，是看人的那种睁。而它看的方向，是我家。
>
> 我必须在明天天亮前搞清楚它想要什么。因为上一个被尸体盯上的入殓师，三天后变成了尸体。

## 标签（5-8 个）

标签 = 平台内搜索和推荐的入口。按优先级排列：

### 必选标签（3 个）
- `#短篇` `#第一人称` `#[题材标签]`（如 悬疑/情感/奇幻/现实主义/复仇）

### 推荐标签（2-5 个）
- 情绪标签：`#虐心` `#细思极恐` `#治愈` `#爽文`
- 类型标签：`#推理` `#反转` `#暗恋` `#重生`
- 知乎特色：`#盐选` `#真实经历`（如故事有真实感包装）
- 热点关联：如与社会热点相关，可挂相关话题标签

---

## 投稿路径

### 站内投稿
1. **知乎后台 → 创作中心 → 发布文章**：基础路径，适合所有短篇
2. **盐选专栏投稿**：需达到一定创作等级和内容质量，编辑审核
3. **圆桌/话题活动投稿**：关注「知乎圆桌」和各类征文活动，按活动规则投稿

### 文件命名建议
```
[作者]_[标题]_[字数]_[日期].md
例：by3_入殓师的第十一年_18000字_20260705.md
```

### 排版建议
- 段落短：手机端每段 ≤3 行
- 空行分段：段落之间一个空行
- 对话分行：长对话独立成段
- 关键句加粗：每节 1-2 处加粗，帮助手机阅读时抓重点
