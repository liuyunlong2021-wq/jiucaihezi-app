# 建制阶段（阶段0-4）

## 阶段 0：对标拆解 → 节拍索引（可选但推荐）

**目标**：选一篇知乎高赞短篇，拆成节拍索引而非线性笔记。

**流程**：
1. 用户提供一篇知乎高赞短篇链接/标题/正文
2. 按节拆解：每节的催化事件、反转类型、钩子模式
3. ★ 输出**节拍索引**而非按节号堆砌——按情景标签组织（冲突类型/反转类型/情绪类型）
4. 写入 `wiki/参考/对标作品/节拍索引.md`（格式见 `../JC-Wiki-记忆压缩/references/小说催化提取.md`）

**旧方式**：`wiki/剧情/对标拆解.md`（按节号线性笔记——翻找低效）
**新方式**：`wiki/参考/对标作品/节拍索引.md`（按情景标签——写作时瞬间检索）

## ★ 阶段 2.5：人物原型激发（新增）

**时机**：阶段2核心梗确认后，阶段3角色设计前。

**目标**：从世界经典角色库为「我」选择一个叙事引擎——不只是「什么样的人」，而是「这个人的决策算法如何自然产生冲突」。

详见 `../JC-短剧-世界模型/references/人物原型激发.md`。短故事适配：

```
我需要为你的「我」找一个叙事引擎——

基于你的核心梗（[简述]）和文风（[简述]），
我扫描了几个最适合第一人称叙事的原型：

### A：费马式叙述者
引擎：技术狂热×对世俗漠不关心。「我」对某件事有极致的专注，
对人际关系天然迟钝——这种「钝」本身就是冲突源。
适合：悬疑/推理/专业领域故事

### B：堂吉诃德式叙述者
引擎：理想主义到偏执。「我」坚信一个在别人看来荒谬的信念，
现实越打脸信念越坚定。
适合：荒诞/黑色幽默/社会讽刺

### C：盖茨比式叙述者
引擎：旁观者+参与者。「我」见证了一个传奇人物的崛起和毁灭，
同时「我」自己也被卷入了这场漩涡。
适合：情感/成长/时代故事

### D：自定义/融合
```

用户选定后 → 生成人物小传（🔒锁定区+🌱生长区）→ 落袋 `wiki/角色/角色名.md`。

**第一人称特殊规则**：
- 原型引擎不仅驱动角色的行为，还驱动**叙述语气**——费马式叙述者说话的方式和盖茨比式完全不同
- 决策算法 = 「我」在压力下怎么想、怎么做、怎么说——这三者必须统一
- 致命弱点 = 第一人称的最大魅力——读者看到「我」的盲区比「我」自己更清楚

## 阶段 1：选题定调

**目标**：结构化抽取用户需求 + 3-5 个选题方案。

**交互模板**：
```
请告诉我：
1. 题材偏好：[悬疑/情感/惊悚/奇幻/现实主义/复仇/成长…]
2. 情绪基调：[冷/暖/压抑/燃…]
3. 目标字数（默认15000-25000）
4. 有没有想避开的话题？
```

用户回复后，输出 3-5 个选题方案，每个含：
- 标题草案
- 一句话主线
- **爆款四要素映射**：话题性/设定反差/主线清晰度/开篇钩子
- 预估字数 + 风险提示

推荐 1 个最优选题，用户确认。

**确认后 → 立即进入阶段 1.5 文风锁定**：

```
阶段1.5 文风锁定流程：
1.5a 文风分类（悬疑冷峻/温情治愈/黑色幽默/社会派纪实/诗意文学/自定义）
1.5b 对标作家+作品推荐（A/B/C/D/E，模型从训练数据搜索）
1.5c 五维文风DNA锁定 → 写入 wiki/世界/文风.md
```

详细协议见 `references/文风分析框架.md`。文风锁是一切后续的基础。

**产出**：`wiki/剧情/选题方案.md` + `wiki/世界/文风.md`

## 阶段 2：核心梗（文风约束）

**目标**：把选题提炼为 Logline + 输出选题专属禁区。

**Logline 公式**：
```
当一个[极致角色]遇到[催化剂]，
必须在[两难选择]中[做什么]，
否则[后果]。
```

**产出**：`wiki/剧情/核心梗.md`

## 阶段 3：角色设计（文风约束）

**目标**：建立主角「我」和配角（≤5个）的完整档案。

**文风约束**：主角「我」的叙述者声音（叙述距离/不可靠程度/语癖）必须匹配 `wiki/世界/文风.md` 的「叙述者声音 DNA」。

**主角「我」**：极致维度 + 内核动机 + 第一人称叙事特质 + 角色弧光 + 压力曲线
**配角**（≤5）：每人身份/欲望/恐惧/关系

**产出**：
- `wiki/角色/主角名.md`
- `wiki/角色/配角名.md`（每个配角）
- `wiki/世界/关系.md`

---

## 阶段 4：大纲（文风约束）

**目标**：10节结构 + 节奏曲线 + 伏笔布局。

**文风约束**：节奏分配（开篇20%/攀升30%/高潮30%/收束20%）需匹配 `wiki/世界/文风.md` 的「叙事节奏 DNA」和「悬念 DNA」。

按知乎节奏模型拆 10 节：
- 每节：目标/冲突/动作/信息增量/悬念 + 字数预估 + 压力值目标
- 标注 3 个关键转折点
- 列出伏笔清单（埋在哪节/在哪节回收）

**产出**：
- `wiki/剧情/大纲.md`
- `wiki/世界/设定.md`
- `wiki/剧情/悬念.md`
