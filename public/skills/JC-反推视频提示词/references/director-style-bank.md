# 视觉风格参考库

> 反推视频时，对照此库识别并描述视频的视觉风格。不强制匹配——如果视频风格不在库里，直接用维度模板自由描述。

## 维度模板

描述任何视觉风格都走这 6 个维度：

```
- 画幅比例：16:9 / 9:16 / 2.39:1 / 4:3 / 1:1
- 光影语言：主光源方向 + 光质(硬/柔) + 色温 + 反差(高/中/低)
- 调色板：主色 + 辅色 + 点缀色 + 饱和度 + 色调(暖/冷/中性)
- 剪辑节奏：秒/镜 + 碎切/长镜/变速
- 机位习惯：常用景别 + 角度偏好 + 运动方式 + 构图特征
- 质感：胶片颗粒/数字锐利/手持晃动/柔焦/粗粝/光滑
```

## 参考案例

以下案例展示维度模板怎么填。反推时参考，但不强制匹配。

### 案例 1：情绪暖调
- 画幅比例：16:9
- 光影语言：暖橘台灯侧光 + 柔光 + 3200K + 中反差
- 调色板：琥珀 #c4a35a + 青灰 #4a5568 + 霓虹品红 · 中高饱和 · 暖调
- 剪辑节奏：慢推慢拉，3-5秒/镜，情绪特写与留白优先
- 机位习惯：框中框构图、长焦空间压缩、缓慢推轨、人物置于门框/窗格内
- 质感：浓艳胶片颗粒

### 案例 2：冷峻精密
- 画幅比例：16:9
- 光影语言：对称冷顶光 + 硬光 + 5600K + 高反差
- 调色板：青灰 #2d3748 + 冷白 #e2e8f0 + 暗红 · 低饱和 · 冷调
- 剪辑节奏：精确中快剪，2-4秒/镜
- 机位习惯：对称构图、稳定运动、审讯式正视、精确推镜
- 质感：数字锐利无颗粒

### 案例 3：自然写实
- 画幅比例：16:9
- 光影语言：自然窗光 + 柔光 + 4500K + 低反差
- 调色板：暖灰 #b0a89c + 米白 #f5f0e8 + 褪色蓝 · 低饱和 · 中性调
- 剪辑节奏：生活流长镜，4-8秒/镜，留白多
- 机位习惯：固定机位旁观、低机位平视、不剪切反应
- 质感：手持生活感

### 案例 4：糖果形式感
- 画幅比例：16:9
- 光影语言：正中心平光 + 柔光 + 5500K + 低反差
- 调色板：粉 #f0a0b0 + 蓝 #60a0d0 + 黄 #f0d060 · 高饱和 · 中性调
- 剪辑节奏：匀速横移，2-4秒/镜
- 机位习惯：绝对对称、中心构图、正前横移、俯视俯瞰
- 质感：舞台剧光滑质感

### 案例 5：宏大史诗
- 画幅比例：2.39:1
- 光影语言：自然硬光 + 硬光 + 5600K + 大光比
- 调色板：冷蓝 #2c5282 + 深灰 #1a202c + 暖橙 · 低饱和 · 冷调
- 剪辑节奏：沉稳长镜，5-10秒/镜，静默与紧迫交替
- 机位习惯：IMAX 广角、宏观与微观对切、沉默长留
- 质感：IMAX 胶片颗粒

### 案例 6：逆光青春
- 画幅比例：16:9
- 光影语言：逆光柔焦 + 柔光 + 5000K + 低反差
- 调色板：青白 #d0dce4 + 淡粉 #f0d8d8 + 雪白 · 低饱和 · 冷调
- 剪辑节奏：缓慢抒情，4-7秒/镜，留白与叠化优先
- 机位习惯：逆光侧拍、柔焦、长焦凝视、空镜留白
- 质感：胶片颗粒柔焦

### 案例 7：手持粗粝
- 画幅比例：16:9
- 光影语言：失真手持光 + 混合 + 不均色温 + 高反差
- 调色板：灰绿 #6b7a6b + 暗褐 #4a3a2a + 惨白 · 低饱和 · 冷调
- 剪辑节奏：晃动手持，2-4秒/镜，不安定感
- 机位习惯：手持跟拍、失焦转清晰、浅景深特写、不完整构图
- 质感：粗颗粒手持

### 案例 8：高饱和民俗
- 画幅比例：16:9 / 2.39:1
- 光影语言：正午硬光 + 硬光 + 5500K + 大光比
- 调色板：红 #cc2222 + 黄 #ddaa00 + 土褐 · 高饱和 · 暖调
- 剪辑节奏：粗放长镜与群像，3-6秒/镜
- 机位习惯：大远景群像、仰拍英雄、自然光硬朗
- 质感：浓烈胶片粗颗粒
