# 动漫视觉风格参考库

> 反推动漫/动画视频时，对照此库识别并描述视觉风格。走 6 维度模板，不强制匹配。

## 维度模板

```
- 画幅比例：16:9 / 9:16 / 4:3
- 作画风格：平涂 cel / 厚涂 / 3D CG / 三渲二 / 水墨 / 剪纸
- 上色与光影：上色方式 + 高光/阴影处理 + 色温 + 反差
- 调色板：主色 + 辅色 + 点缀色 + 饱和度
- 演出节奏：秒/镜 + 碎切/长镜 + 动作张数
- 镜头特征：常用景别 + 运动方式 + 特效手法
```

## 参考案例

### 现代热血 cel
- 画幅比例：16:9
- 作画风格：平涂 cel + 粗线稿 + 动态透视
- 上色与光影：heavy cel shading + 暗调底色 + 霓虹 rim-light + 极高反差
- 调色板：深蓝 #1a1a3a + 青紫 #6040a0 + 绯红溅色 · 高饱和 · 冷调
- 演出节奏：极快，1-3秒/镜，流畅武打长镜优先
- 镜头特征：motion-blur 拖影、速度线、构图切割、EDM 节奏剪辑

### 经典少年热血
- 画幅比例：16:9
- 作画风格：平涂 cel + 清晰线稿
- 上色与光影：明亮平涂 + 暖调主色 + 低反差
- 调色板：橙红 + 海蓝 + 金黄 · 高饱和 · 暖调
- 演出节奏：中快，2-4秒/镜，变身/觉醒长镜优先
- 镜头特征：速度线、集中线、夸张表情、伸缩变形

### CG 电影写实
- 画幅比例：16:9 / 2.39:1
- 作画风格：3D CG 渲染
- 上色与光影：CGI 电影光 + 冷蓝高光 + 暖橙补光 + 高反差
- 调色板：冷蓝 + 暖橙 + 金属灰 · 中高饱和 · 冷调
- 演出节奏：电影感慢镜与高速战斗交替，3-6秒/镜
- 镜头特征：体积粒子特效、镜头光晕、英雄落地慢镜、刀光拖影

### 国产 3D 仙侠
- 画幅比例：16:9
- 作画风格：国产 3D 写实
- 上色与光影：仙侠冷光 + 法宝金光 + 中高反差

- 演出节奏：中快，2-4秒/镜，御剑飞行长镜优先
- 镜头特征：3D 运镜、灵气粒子、山水意境、写实材质

### 中国动画学派 · 工笔重彩

- 镜头特征：铁线描轮廓、云纹图案、京剧脸谱化、平面装饰构图
