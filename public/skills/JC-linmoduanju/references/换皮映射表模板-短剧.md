# 换皮映射表模板（短剧版）

> 短剧/漫剧临摹专用。这是全篇一致性的**事实源**。首次进入从范本前文抽取初版，逐集遇新元素即时补登，全表查重防撞车、防一名多译。
>
> 短剧版比小说版多强调**第5张表（桥段/道具/动作/反转映射）**——这是防"一眼认出临摹自哪部"的命门。

## 换皮三层边界（先认边界，再查表）

```
【骨·必须保留】抽象层：剧情走向、人物命运与结局、人物关系、爽点节奏、付费卡点位置、主题、基调
【桥段·保功能换实现】桥段层：每个场次/反转"要达成的戏剧作用"保留，用什么桥段/道具/动作去达成 → 全换（见第5表）
【皮·必须全换】表层：人名、地名、专名、台词措辞、象征符号、场景表面细节
```
口诀：临摹学的是"这一场要干什么"，不是"用什么桥段干"。

### 5. 桥段/道具/动作/反转映射（★ 短剧临摹命门：保功能·换实现）

> 左列是范本标志性的场次设计及其**戏剧功能**，右列是换皮后用什么**完全不同的实现**达成同一功能。
> 红线：右列实现必须和范本"长得不一样"，不能只换名词照搬。

| 范本桥段（戏剧功能） | 新作实现（功能同·面貌全换） | 集 |
|----------------------|------------------------------|:--:|
| 例：当众泼酒打脸羞辱主角（建立屈辱、埋反击动机） | 例：当众撕毁主角的设计稿/合同 | 1 |
| 例：扔出一张银行卡甩脸（炫富压制） | 例：甩出一串车钥匙/一份房产证 | 3 |
| 例：雨中下跪挽留（情绪高潮卡点） | 例：医院走廊追到电梯关门前一刻 | 8 |

- 这张表管的是范本最显眼的"名场面"。短剧的打脸/反转/卡点桥段往往是观众记住一部剧的关键，**照搬只换名字=抄袭**，必须换实现。

## 使用要点

- **首次抽取**：通读范本+确认换皮方案后填新名。范本没出现的不硬编。
- **逐集即时登记**：每集遇新元素先登记再写。登记时全表扫：新名是否重复（撞车→换）？旧名是否已登记过别的新名（不一致→用旧的）？
- **改名同步**：用户中途改名，立刻更新映射表，提醒"前面已改集里的名字也要同步"（可用 JC-yizhixing 巡检批量核查）。
- **第5表是短剧专属重点**：小说临摹也讲桥段换皮，但短剧的"名场面/卡点"更显眼、更易被认出，第5表务必建细。
- **一致性是命脉**：长剧换皮翻车 90% 因映射不一致——漏改、改串、撞车，或桥段照搬。映射表维护到位，作品才像原创。
