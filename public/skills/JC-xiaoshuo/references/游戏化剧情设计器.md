# SKILL-05: 游戏化剧情设计器

## 核心职责

用游戏思维设计爽点分布，确保奖励递进

## 卡牌分级系统

| 级别 | 频率 | 爽感强度 | 示例 |
|------|------|----------|------|
| **R卡** | 每2-3章 | ★ | 小打脸、被夸、小胜利 |
| **SR卡** | 每10-15章 | ★★★ | 马甲部分揭露、中型打脸 |
| **SSR卡** | 每50-80章 | ★★★★★ | 马甲完全掉落、终极打脸 |

## 地图设计思维

### 新手村（1-50章）

```
开放地图：豪门+医院
掉落奖励：R卡×5 + SR卡×1
Boss：假千金、势利豪门
通关奖励：豪门态度转变
```

### 主城（51-150章）

```
开放地图：医学界+商界
掉落奖励：SR卡×3 + R卡×10
Boss：商业对手、医学界质疑者
通关奖励：M医生身份揭露
```

### 王都（151-280章）

```
开放地图：国际舞台
掉落奖励：SSR卡×2 + SR卡×5
Boss：国际挑战者
通关奖励：国际认可
```

### 最终副本（281-380章）

```
开放地图：国际医学大会
掉落奖励：SSR卡×1（集大成）
Boss：终极质疑+自我突破
通关奖励：终极HE
```

## 奖励递进原则

```
禁止重复奖励：
❌ 第20章打脸A，第40章打脸B（重复）
✅ 第20章打脸A，第40章打脸A+获得资源（递进）

奖励叠加：
后面的奖励 = 前面的奖励 + 新奖励
```

## 检查清单

- [ ] 每个阶段的SSR奖励是什么？
- [ ] R卡分布是否足够密集（每2-3章）？
- [ ] 奖励是否有递进（不是重复）？
- [ ] 为了拿SSR，是否有前置任务？
