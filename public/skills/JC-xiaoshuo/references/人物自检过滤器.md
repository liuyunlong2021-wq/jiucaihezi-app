# SKILL-05: 人物自检过滤器 (Character QC)

> 防止制造出 "便利贴人物" (Cardboard Characters)。

---

## 核心功能

运行基于《你的剧本逊毙了！》的自检程序，对已生成的人物进行压力测试。

---

## 检查清单 (The Checklist)

### 1. "他在想什么"测试 (The Want Test)

* **Check**: 你能否清晰地写出主角在每一场戏的"目的" (Want)？
* **Rule**: 主角在全片有超级目标 (Super Objective)，在每场戏有即时目标 (Scene Objective)。如果找不到，人物就是模糊的。

### 2. 名字检查 (Name Check)

* **Check**:
  * 首字母是否不同？(避免 John, Jack, Jane 同时出现)。
  * 名字是否发音相近？(避免 Tim, Tom)。
  * 名字是否暗示了性格？(Valid: "Hammer" for a tough guy; Invalid: "Daisy" for a killer unless ironic).

### 3. 声音测试 (Voice Test)

* **Check**: 遮住名字，能否只看对白就分辨出是谁在说话？
* **Rule**: 如果所有人都像 AI（或作者本人）一样说话，Fail。
* **Fix**: 给每个角色分配特定的句式长度、口头禅、受教育程度标记。

### 4. 坏蛋检查 (Villain Test)

* **Check**: 你的坏蛋是否觉得自己才是英雄？
* **Rule**: 坏蛋必须有自己的逻辑和正义感。不能为了坏而坏。
* **Fix**: 给反派一个合理的动机（如：为了秩序、为了拯救世界、为了生存）。

### 5. 主动性检查 (Active Hero Test)

* **Check**: 故事是主角驱动的，还是事情发生在主角身上的？
* **Rule**: 主角必须是引擎。如果主角一直被动反应，Fail。

---

## 执行指令

**Input**:
完整的人物设定。

**Process**:

1. Run Checklist 1-5.
2. **Critique**: 指出不合格的项目。
3. **Suggest**: 给出修改方案（如：改名、重写对白风格、增强反派动机）。

**Output**:

* **自检报告**: [Pass/Fail细节]
* **优化建议**: ...
