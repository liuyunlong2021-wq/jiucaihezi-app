# 短故事结构模板

> 不同题材的短故事有不同的结构骨架。以下提供 5 种经典模板，可单独使用或混合。

## 模板一：倒计时型（悬疑/惊悚/动作）

**核心**：一个明确的时间限制驱动全篇。

```
结构：
开篇 → 「我」知道了倒计时（还剩X天/X小时）
铺垫 → 第一次尝试解决，失败，倒计时缩短
攀升 → 每一次尝试都让情况变得更糟
高潮 → 最后一刻，意想不到的解决（或不可逆转的失去）
尾声 → 尘埃落定后的「我」
```

**字数分配**（以 20000 字为例）：
- 开篇 2000 字：倒计时设定 + 第一次冲击
- 铺垫 4000 字：两次失败尝试
- 攀升 8000 字：情况恶化 + 盟友出现或背叛 + 最大挫折
- 高潮 4000 字：终极对决/抉择
- 尾声 2000 字：余波

**示例题材**：被困密室、追查失踪者、阻止灾难、竞赛/挑战

---

## 模板二：关系弧线型（情感/成长/文艺）

**核心**：一段关系从建立到变化到终结（或超越）的完整弧线。

```
结构：
开篇 → 「我」遇见了那个会改变一切的人
铺垫 → 关系建立，温度上升，暗涌初现
攀升 → 裂痕出现 → 试图修补 → 更大的裂痕
高潮 → 关系中的决战时刻（分手/和解/告别/在一起）
尾声 → 「我」独自面对之后的日子
```

**字数分配**（以 20000 字为例）：
- 开篇 2000 字：相遇 + 第一印象
- 铺垫 4000 字：关系升温 + 甜蜜/温暖场景 + 第一个暗涌
- 攀升 8000 字：矛盾升级 + 尝试挽回 + 决定性事件
- 高潮 4000 字：最终对话/抉择场景
- 尾声 2000 字：「我」的成长/改变

**示例题材**：初恋、离婚、重逢、和解、暗恋、亲情修复

---

## 模板三：揭开真相型（悬疑/心理/现实主义）

**核心**：「我」逐步发现一个被隐藏的真相。

```
结构：
开篇 → 一个异常细节引起了「我」的注意
铺垫 → 调查/追问，收集碎片，每个碎片都让事情更复杂
攀升 → 接近真相时遭遇阻碍/威胁
高潮 → 真相揭晓——往往不是「我」想听到的答案
尾声 → 「我」带着真相（或破碎的幻象）继续生活
```

**字数分配**（以 20000 字为例）：
- 开篇 2000 字：异常发现 + 「我」决定追查
- 铺垫 4000 字：前两个线索的收集和解读
- 攀升 8000 字：线索矛盾 → 阻碍出现 → 误入歧途 → 最终线索
- 高潮 4000 字：真相揭露 + 「我」的反应
- 尾声 2000 字：接受/拒绝真相

**示例题材**：家族秘密、失踪案、身份谜团、超自然现象

---

## 模板四：蜕变型（成长/治愈/励志）

**核心**：「我」经历一次根本性的内在改变。

```
结构：
开篇 → 「我」被困在某种状态中（但不自知或已麻木）
铺垫 → 催化剂打破平衡，「我」被迫面对问题
攀升 → 改变过程中的抗拒、反复、痛苦
高潮 → 最关键的一次选择——选择改变
尾声 → 「我」已然不同
```

**字数分配**（以 20000 字为例）：
- 开篇 2000 字：「我」的困境状态 + 第一次动摇
- 铺垫 4000 字：催化剂 → 尝试改变 → 第一次退缩
- 攀升 8000 字：反复拉锯 → 最黑暗时刻 → 转折点
- 高潮 4000 字：关键选择 + 行动
- 尾声 2000 字：新的「我」

**示例题材**：走出创伤、戒断、重新开始、自我和解、追梦

---

## 模板五：闭环型（文学/实验性/哲思）

**核心**：首尾呼应，形成一个完美的闭环。不是线性叙事，而是循环。

```
结构：
开篇 → 一个画面/一句话/一个场景
发展 → 离开这个画面，展开故事
回环 → 在结尾回到同一个画面——但意义已经完全不同
```

**技巧**：
- 开篇和结尾用同一句话，但读者读完后的理解完全不同
- 开篇的一个物品、一个动作，在结尾再次出现，被赋予了新的意义
- 「我」在开头做的事，在结尾重新做了一遍——但心态已经彻底改变

**示例**：
```
开篇：「我叫林安。这是我在这个世界上的第二十七年零三个月。也是最后一天。」
……（故事展开）……
结尾：「我叫林安。这是我在这个世界上的第二十七年零三个月。也是第一天。」
```

**字数分配**：与传统结构相同，但首尾各留 2000 字做闭环设计。

---

## 使用原则

1. **不要死套模板**：模板是骨架，不是牢笼。混合使用最常见（如倒计时型+关系弧线型 = 被困期间的情感故事）
2. **字数比例灵活调整**：20000 字的比例和 10000 字不同——越短则开篇越要快
3. **短故事可以省略「铺垫」**：如果故事只有 6000-8000 字，开篇 1000 字直接进冲突，铺垫揉进攀升里
4. **高潮永远是情绪最高点**：不管什么模板，高潮段的情绪强度必须是全篇最强
