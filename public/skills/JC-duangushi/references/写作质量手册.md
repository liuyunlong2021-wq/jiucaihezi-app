# 写作质量手册

## 知乎短篇节奏模型

```
S1-S2（开篇）    20%   前50字钩子 → 建立「我」的世界 + 催化剂触发
S3-S5（攀升）    30%   压力持续升级，每节一个小转折
S6-S8（高潮区）  30%   压力爆发到顶点，最大反转/抉择/对抗
S9-S10（收束）   20%   伏笔回收 → 情绪回响 → 落在一个有分量的画面上
```

## 节奏铁律

- **开篇 50 字定生死**：必须有冲突/悬念/反差。禁止"我叫XX，今年XX岁"式自述开场
- **每节必推进主线**：一节 = 目标 + 冲突 + 动作 + 信息增量 + 悬念
- **每节末留钩子**：让读者想点"下一节"
- **中段不塌腰**：S4-S6 是最危险的塌腰区，用催化剂引爆新冲突
- **收尾有回响**：不一定要大团圆或悲剧，但要有让读者合上之后还在想的画面

## 第一人称写作规则

### 情绪具象化
- ❌ "我很紧张" → ✅ "我的拇指指甲掐进了食指的肉里"
- ❌ "我感到害怕" → ✅ "走廊的灯一闪一灭，我的脚步越来越慢"
- ❌ "我很难过" → ✅ "我看着那杯水，看了很久，直到它完全凉了"

### 对话规则
- 每句对话必须有功能：贴人设、推剧情、调情绪
- 对话可分行也可嵌入段落
- 格式：`"你为什么要回来？"我听见自己的声音在发抖。`

## 钩子法则

每节末尾四种钩子：

| 类型 | 说明 | 示例 |
|------|------|------|
| 悬念钩子 | 信息差，读者想知道后续 | "门开了。站在门外的，是已经死了三年的人。" |
| 情绪钩子 | 强烈情绪定格 | "我没有哭。我只是站在那里，看着她的车消失在雨里。" |
| 反转钩子 | 认知被颠覆 | "那一刻我才明白，她从来没有爱过我。" |
| 行动钩子 | 角色即将行动 | "我拨通了那个号码。嘟——嘟——" |

## 写作格式

```markdown
## S1
我睁开眼的时候，天花板上有一只蜘蛛。
（正文……）

---

> 后来我想起那个下午，才发现她早就给了我答案。

## S2
门铃响的时候，我正在洗掉手上的血。
```

- 段落标题用「S1」「S2」…「S10」
- `>` 用于「我」的内心独白/金句/反思
- 对话嵌入段落或独立成行

## 写作铁律

### ✅ 必做
- 前50字有冲突/悬念/反差
- 第一人称到底，绝不跳上帝视角
- 主线单一，一件事到底
- 每节推进主线 + 留悬念
- 情绪具象化，用身体反应代替心理描述
- 对话有功能
- 细节精确（时间/地点/称谓/流程）
- 收尾有回响

### 🚫 禁止
- "我叫XX，今年XX岁"无冲突开场
- 上帝视角：写别人的内心活动
- 人称漂移：变成第三人称
- 「我感到」「我觉得」满天飞
- 塌腰：连续2000字以上压力没变化
- 配角过多（≤5个）
- 主线发散
- 烂尾：伏笔不回收
- 大段无功能抒情
- 形容词堆砌

## 自检清单

- [ ] 前50字有冲突/悬念/反差
- [ ] 第一人称全程一致，无上帝视角
- [ ] 主线单一清晰
- [ ] 每节推进主线 + 留钩子
- [ ] 情绪具象化，用身体反应
- [ ] 对话有功能
- [ ] 伏笔全部回收
- [ ] 字数在目标范围

## 博伊尔检查·短故事适配版（每节写完后 5 秒快检）

> 来源：威廉·M·埃克斯《你的剧本逊毙了！》#30 #32 #34 + duanju 升级经验

### 1. 叙事转向

**每节结束时的信息状态必须和开始时不同。** 信息变了 = 故事推进了。

```
开始状态：读者知道什么？
      ↓
本节事件：什么新信息被揭示/什么关系被改变？
      ↓
结束状态：读者现在知道了什么新的？
```

❌ 开始：知道 A 死了 → 结束：还是只知道 A 死了（没转向）
✅ 开始：知道 A 死了 → 本节发现 A 死前见过 B → 结束：读者知道 B 和 A 的死有关（转向了）

### 2. 鸟食检查

**不要一次倒完所有信息。** 悬念不是信息量的问题，是释放节奏的问题。

```
鸟食原则：这一节有没有一次性揭示太多？
    → 能不能把一个大信息拆成 2-3 次释放？
    → 最重要的信息是不是留到了本节最后？
```

❌ S3 同时揭晓：凶手是谁 + 动机 + 作案手法
✅ S3：发现凶手身份 → S5：揭示动机 → S8：还原作案手法

### 3. 冲突升级检查

**每节的冲突必须比上一节更紧。** 读者应该感觉螺丝被越拧越紧。

❌ S4 冲突 = S3 冲突的换个场景重演一次
✅ S4：S3 的问题不但没解决，还引发了更严重的后果

## 文风驱动写作

> 当 `wiki/世界/文风.md` 存在时生效。叠加在原有规则之上，不是替代。

### 核心原则

**文风 = 这篇故事的「味道」**。由一个具体作家+作品（或多作品杂交）锁定的五维文风 DNA。一旦锁定，语言风格、悬念节奏、叙述者声音、催化剂设计都必须带着这个味道。

### 从五维文风 DNA 推导写作规则

| 从 DNA 的哪个维度 | 推导写作的什么约束 |
|-------------------|-------------------|
| 叙事节奏 DNA | 每节信息密度、段落长度、留白比例 |
| 语言风格 DNA | 句式长短、词汇冷暖、比喻密度、对话叙述比 |
| 悬念 DNA | 伏笔释放节奏、反转类型偏好、钩子密度 |
| 情绪基调 DNA | 每节情绪在基调范围内、情绪曲线形状 |
| 叙述者声音 DNA | 「我」的口吻/语癖/叙述距离、不可靠程度 |

### 催化剂文风驱动原则

阶段5逐节写作的催化剂选项必须是文风驱动的：

- 悬疑冷峻风 → 催化剂优先：新线索/旧案串联/日常裂痕
- 温情治愈风 → 催化剂优先：意外重逢/迟来的道歉/隐瞒的善意
- 黑色幽默风 → 催化剂优先：荒诞误会/身份错位/计划乌龙
- 社会派纪实风 → 催化剂优先：证据浮现/证人翻供/权力施压

### 文风脱轨自检

每节写完后用 3 个问题自检：

1. **如果把这一节放到另一个文风的短篇里，还能用吗？** 如果能→文风不鲜明。
2. **这一节有「只有这个文风才能写出来」的东西吗？** 如果没有→文风没落地。
3. **「我」的叙述口吻是否全程一致？** 如果中途变味→检查文风 DNA 的叙述者声音。
