# 核心机制

## 极致性系统

| 维度 | 红线 | 催化剂示例 |
|------|------|-----------|
| 自由 | 被束缚/被命令 | 被囚禁 |
| 正义 | 目睹不公/被冤枉 | 无辜者被杀 |
| 复仇 | 仇人逍遥/被羞辱 | 亲人被杀 |
| 爱情 | 失去爱人/被背叛 | 爱人被抓 |
| 权力 | 失去地位/被轻视 | 被罢免 |
| 知识 | 真相被隐瞒 | 发现秘密 |
| 生存 | 死亡威胁 | 绝境逃生 |
| 荣誉 | 名声被毁 | 被公开诽谤 |
| 贪婪 | 财富被夺 | 宝藏被发现 |
| 守护 | 要保护的人受威胁 | 亲人被绑架 |

## 压力值系统

| 压力值 | 状态 | 段落功能 |
|--------|------|----------|
| 0-30 | 平静 | 铺垫/背景/喘息 |
| 30-70 | 紧张 | 冲突升级/推进 |
| 70-100 | 爆发 | 高潮/转折/抉择 |

变化规则：触碰红线 +50 / 催化剂触发 +20 / 冲突解决 -20 / 冲突升级 +30 / 两难选择 +15。
压力曲线呈"爬坡型"：开篇中等压力 → 持续攀升 → 高潮爆发 → 尾声回落。

## 催化剂类型

| 类型 | 说明 | 短故事示例 |
|------|------|-----------|
| 外部入侵 | 外部力量打破平衡 | 一封陌生来信、一个不速之客 |
| 内部觉醒 | 角色认知改变 | 「我」发现记忆被篡改 |
| 关系裂变 | 人际突发变化 | 最信任的人背叛「我」 |
| 世界突变 | 环境剧烈变化 | 「我」被困在时间循环里 |

**催化剂设计必须读取 `wiki/世界/文风.md`**。2-3 个选项需带文风风味。详见 `references/写作质量手册.md`「文风驱动写作」。

## 第一人称叙事规则

### 「我」的层次

| 层次 | 说明 |
|------|------|
| 视角层 | 「我」只能看到/听到/闻到/感受到什么 |
| 认知层 | 「我」如何理解和判断眼前的事 |
| 语气层 | 「我」用什么词汇、句式、节奏讲述 |

### 第一人称铁律

1. 「我」不知道的就是不知道：不能跳上帝视角
2. 「我」的情绪即读者情绪：不写"我很害怕"，写「我的手在发抖」
3. 「我」的声音必须独一无二
4. 叙述距离可变：事后回顾 vs 即时经历

### 不可靠叙述者

| 类型 | 说明 | 适合题材 |
|------|------|----------|
| 无知型 | 「我」不知道真相，和读者一起发现 | 悬疑/惊悚 |
| 隐瞒型 | 「我」知道但不说，逐步透露 | 复仇/反转 |
| 自欺型 | 「我」在骗自己，读者察觉到裂痕 | 情感/成长 |
| 疯狂型 | 「我」的精神状态扭曲现实 | 心理惊悚 |
