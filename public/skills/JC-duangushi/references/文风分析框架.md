# 文风分析框架（Style Analysis Framework）

> **核心哲学**：LLM 训练数据覆盖人类所有文学作品。本文件不提供封闭作家列表——它是一套**文风分析协议**，教模型如何当场拆解**用户提到的任何作家/作品**的文风 DNA。

---

## 一、为什么不用作家列表

- LLM 天然知道蔡骏的悬疑节奏、紫金陈的社会派冷峻、priest 的诗意对白、东野圭吾的反转密度……不需要我们列。
- 列 12 个条目 = 限制用户选择范围 = 浪费模型知识。
- 本文件的价值在**分析方法**，不在**答案库**。

## 二、五维文风拆解协议

对用户提到的任何参考作家/作品，当场拆解为 5 个维度。这是后续一切（催化剂/反转/语言/钩子）的驱动源。

### 维度 1：叙事节奏 DNA

| 问自己 | 产出 |
|--------|------|
| 每 3000 字的信息密度？ | 高密度（每段推进）/ 中等 / 留白型 |
| 章节推进速度？ | 快推（每节一个大事件）/ 稳步 / 慢炖蓄力 |
| 铺垫和爆发的比例？ | 铺垫:爆发 ≈ X:Y |
| 知乎十节适配（关键）| S1-S2 开篇需要多快？信息在十节内如何分布？ |

### 维度 2：语言风格 DNA

| 问自己 | 产出 |
|--------|------|
| 句式特征？ | 短句为主（≤15字）/ 长短交替 / 绵长描写 |
| 词汇冷暖？ | 冷峻克制 / 温暖细腻 / 粗粝生猛 / 诗意华丽 |
| 比喻使用频率？ | 密集（每段都有）/ 偶用点睛 / 几乎不用 |
| 对话与叙述比例？ | 对话为主 / 叙述为主 / 均衡 |

### 维度 3：悬念 DNA

| 问自己 | 产出 |
|--------|------|
| 伏笔密度？ | 每节至少 1 个伏笔 / 每 2-3 节一个 / 稀疏 |
| 信息释放节奏？ | 一次性揭示 / 分层释放 / 藏到最后 |
| 反转类型偏好？ | 认知反转 / 身份反转 / 关系反转 / 真相反转 |
| 反转规模？ | 小反转（每节）/ 中反转（每 3-4 节）/ 大反转（收束段） |

### 维度 4：情绪基调 DNA

| 问自己 | 产出 |
|--------|------|
| 基调情绪是什么？ | 冷/暖/压抑/燃/荒诞/诗意/锋利 |
| 情绪曲线形状？ | 持续攀升型 / 过山车型 / 渐暖型 / 渐冷型 |
| 情绪强度？ | 克制内敛 / 中等 / 泪点密集 |

### 维度 5：叙述者声音 DNA（第一人称核心）

| 问自己 | 产出 |
|--------|------|
| 叙述距离？ | 即时经历（正在发生）/ 事后回顾（"后来我才知道"）/ 交替 |
| 不可靠叙述者类型？ | 无知型 / 隐瞒型 / 自欺型 / 疯狂型 / 可靠型 |
| 语癖/口吻？ | 冷嘲 / 自贬 / 克制 / 碎碎念 / 干脆利落 |
| 「我」的独特声音？ | 一句话描述「我」的叙述风格 |

## 三、校准示例（不是菜单，是格式锚点）

### 示例 A：蔡骏《天机》

```
## 文风 DNA — 蔡骏《天机》

- 叙事节奏：高密度，每章一个新线索。铺垫:爆发 = 3:7。
  知乎适配：S1 直接抛出超自然现象，不给读者喘息机会。
- 语言风格：冷峻短句为主，≤15字。词汇偏冷，比喻偶用点睛。
- 悬念：伏笔密集，信息分层释放。反转类型以真相反转为主。
- 情绪基调：压抑 + 惊悚。情绪曲线持续攀升，不给温暖喘息。
- 叙述者声音：隐瞒型不可靠叙述者。「我」知道更多但慢慢说。
  口吻：克制、冷静，用平淡语气描述恐怖事件。

### 催化剂设计原则
优先：超自然现象递进、日常中的异常、你以为的真相是假的
避免：温情日常、搞笑桥段、明确的HE暗示

### 标志性元素可借用
平凡日常中的裂缝逐渐扩大、历史与当下的隐秘联系、
「你以为结束了，其实才刚刚开始」式章节收尾
```

### 示例 B：紫金陈《长夜难明》

```
## 文风 DNA — 紫金陈《长夜难明》

- 叙事节奏：稳步推进，每节一个证据/一个阻力。铺垫:爆发 = 5:5。
  知乎适配：S1 从结果倒叙切入（"十年后我才知道那天意味着什么"）。
- 语言风格：社会派纪实风格，平实克制。比喻极少，靠事实本身制造冲击。
- 悬念：伏笔中等密度，信息逐步释放。反转类型：认知反转为主（"原来他才是…"）。
- 情绪基调：压抑→愤怒→最后一丝温暖。情绪曲线先抑后扬，收束有回响。
- 叙述者声音：事后回顾型可靠叙述者。「我」在讲述一个已经发生的故事。
  口吻：克制但不冷漠，偶尔流露愤怒与无力。

### 催化剂设计原则
优先：新证据出现→被销毁、找到证人→证人翻供、权力施加压力
避免：超自然元素、巧合过多、主角光环式翻盘

### 标志性元素可借用
「小人物对抗庞大机器」的无力感与坚持、
每推进一步就遭遇更大阻力、「长夜将尽」式的收束情绪
```

### 示例 C：priest《默读》

```
## 文风 DNA — priest《默读》

- 叙事节奏：中速推进，案件与人物交替。铺垫:爆发 = 4:6。
  知乎适配：S1 开场直接进罪案现场，用「我」的视角观察。
- 语言风格：长短句交替，对话占比高（~40%）。语言干脆利落，偶有诗意比喻。
  对话风格：你来我往、略带机锋、冷不丁一句金句。
- 悬念：伏笔中等密度，案件为主线、人物关系为暗线。反转：关系反转 + 真相反转。
- 情绪基调：冷中带暖。罪案的冷 × 人物关系的暖。
- 叙述者声音：即时经历型可靠叙述者。口吻：利落、不废话、偶尔冷幽默。

### 催化剂设计原则
优先：新案件线索出现、旧案与当下串联、人物关系突然变化
避免：大段心理描写、拖沓日常、纯搞笑

### 标志性元素可借用
双线叙事（案件线 + 人物线）、每案结尾的"你以为破了→更大的局"、
对话中的人物弧光、「黑暗中的一缕光」式情绪收束
```

## 四、杂交融合协议

用户选 A+B 杂交时，用三步法：

1. **找交集**：A 和 B 有什么共同 DNA？例：蔡骏的「信息分层释放」+ 紫金陈的「认知反转」→ 交集 = 逐步揭示真相的节奏感。

2. **互补**：A 有什么 B 没有？例：蔡骏的「超自然悬疑」+ 紫金陈的「社会派纪实」→ 超自然事件的社会派写法。不是鬼怪，是「让人以为有鬼怪，其实是人在作恶」。

3. **防冲突**：检查互斥维度。例：priest 的「诗意比喻」+ 紫金陈的「平实纪实」→ 可能打架。需明确**主文风**和**辅文风**。

### 杂交输出格式

```
## 杂交文风 DNA — A《X》× B《Y》

- 主文风：[A 或 B]，理由：[为什么选它做主]
- 辅文风：[另一个]，注入：[具体注入什么元素]
- 化学反应点：[A 和 B 碰撞产生的独特东西]
- 叙事节奏：[融合后]
- 语言风格：[融合后]
- 悬念：[融合后]
- 情绪基调：[融合后]
- 叙述者声音：[融合后]

### 催化剂设计原则
[融合后的催化剂偏好，标注哪些来自 A、哪些来自 B]
```

## 五、知乎十节短篇适配

长篇小说的节奏不能直接套到 10 节短篇。分析完参考作品后必须做适配：

| 维度 | 长篇小说 | 知乎十节适配 |
|------|---------|-------------|
| 开篇速度 | 可以 2-3 章铺垫 | S1 前 50 字必须有钩子 |
| 伏笔周期 | 可以跨 20 章回收 | 伏笔必须在 10 节内回收 |
| 配角深度 | 可以 5-10 个配角慢慢展开 | 配角 ≤5 个，标签化 |
| 信息密度 | 可以稀释 | 每节必须有信息增量 |
| 反转规模 | 可以慢慢铺大反转 | 每节有小反转，S6-S8 大反转 |

## 六、反模式

| ❌ | 为什么错 |
|----|----------|
| 「文笔好」「有画面感」等抽象词 | 无法执行。必须具体到某部作品的句式/节奏/悬念密度 |
| 同时推荐同一个作家的 3 部作品 | 不是真正的差异化 |
| 杂交时两个文风五五开 | 等于没有文风。必须明确主辅 |
| 忽略知乎平台的节奏要求 | 长篇节奏不适合 10 节短篇 |
| 只复制语言风格忽略了悬念设计 | 知乎短篇核心是悬念节奏，不只是文笔 |

## 七、模型自主推荐策略

阶段 1.5 推荐 A/B/C 时，模型应：

1. **读阶段1产出**（题材/情绪基调/目标字数）→ 理解故事底色
2. **从训练数据中搜索** → 找 3 个文风差异明显的作家+作品
3. **当场拆解** → 每部作品出五维文风 DNA（用 §二 协议）
4. **标注适配理由** → 为什么这个文风适合这个选题

**推荐覆盖原则**：3 个选项朝 3 个方向拉开——如一个冷峻悬疑向、一个社会派纪实向、一个诗意文学向。
