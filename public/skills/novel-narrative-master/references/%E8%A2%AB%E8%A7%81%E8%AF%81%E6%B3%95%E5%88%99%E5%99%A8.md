# SKILL-05: 被见证法则器 👥

## 核心职责

**确保主角的高光时刻被充分看见、被反应、被震惊。**
没有反应镜头的装逼，是无效装逼。

## 核心公式

```
完整的爽文叙事 = 做 (Action) + 改变 (Change) + 被见证 (Witness)
```

**示例**：

- **做**：程月施针救人。
- **改变**：病人脸色红润，心跳恢复。
- **被见证**：
  > 院长瞪大了眼睛，眼镜滑落都不自知。
  > "这...这是失传已久的鬼门十三针？！"
  > 周围的实习生们屏住了呼吸，看程月的眼神像在看神仙。
  > 刚才嘲讽的那个医生，此刻脸涨成了猪肝色，恨不得找个地缝钻进去。

## 反应镜头分配

### 1. 打脸场景（反应占比：50%）

主角只需要做一个简单的动作（比如甩出一张卡），重点全在别人的反应。

- **反派反应**：嚣张 → 疑惑 → 惊愕 → 恐惧 → 崩溃
- **路人反应**：看戏 → 震惊 → 崇拜 → 议论纷纷（侧面烘托）
- **大佬反应**：赏识 → 想要结交

### 2. 甜宠场景（反应占比：50%）

主角互动的每一个细节，都要有反馈。

- **对方反应**：脸红、心跳加速、不知所措、眼神躲闪
- **情敌反应**：嫉妒、酸、觉得自己多余
- **助攻反应**：姨母笑、磕到了

## 震惊阶梯（Hierarchy of Shock）

为了让震惊更有层次感，不要总是"倒吸一口凉气"。

1. **生理反应**：瞳孔收缩、手抖、冷汗、失语、腿软。
2. **语言反应**：结巴、语无伦次、大喊大叫、喃喃自语。
3. **行动反应**：后退一步、杯子掉落、跪下、冲上来查看。
4. **心理反应**：世界观崩塌、怀疑人生、后悔莫及。

## 检查清单

### 三问自检

1. **"主角刚才那个那个帅动作，有人看到吗？"**
   - 没人看到 → **安排路人/监控/直播！**

2. **"反应够强烈吗？"**
   - 只是"大家都很惊讶" → **不够！** 描写细节！谁把下巴惊掉了？谁后悔得想哭？

3. **"重要人物（反派/CP）的反应写了吗？"**
   - 他们的反应最值钱，必须给特写。
