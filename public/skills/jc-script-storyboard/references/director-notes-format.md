# 导演工作本输出格式

> 阶段 3 写分镜时读取。

## 格式

```
镜号（独立一行）
导演阐述段落（自然语言，2-6句）
```

每个镜头的阐述是一段导演在监视器前的思考——他在想什么、他要什么、他参考了自己哪部作品的哪个场景。

## 阐述的构成

自然融入以下要素，不生硬罗列：

- **开场画面**：观众看到的第一眼
- **角色表演方向**：演员做什么、什么情绪、身体语言、微表情
- **机位与构图**：为什么这个角度、参考了该导演哪部作品的镜头
- **光影与空间**：光从哪来、色温、阴影、材质质感（下游生图/视频模型的关键锚点）
- **镜头运动与时长**：推拉摇移跟固定、持续秒数
- **音效设计**：环境音、特殊音效、音乐情绪

## 短镜（≤6秒）

```
1-1-1
门被撞开的瞬间——从门口向外拍，走廊荧光灯管的青白冷光先涌入画面，然后是闯入者的剪影。小红在床边猛然回头，她的脸一半在暖橘台灯的琥珀色光晕里，一半陷在暗处。镜头固定，4秒——观众看到的是小红从惊醒到戒备的全过程。参考《花样年华》中周慕云敲门时苏丽珍在门内犹豫的那个构图：人被困在门框里。音效：门撞开的锐响后立刻切入压抑的低频嗡鸣。
```

## 长镜（>6秒）用时间分段

```
1-1-7
0-3秒：镜头从窗外霓虹招牌缓缓摇进房间，百叶窗的青色条纹扫过小红的后背
3-6秒：摇到门口，闯入者的剪影逐渐清晰，走廊荧光灯闪烁，像是心跳
6-10秒：小红站起来，暖橘台灯的光在她脸上晃动，她的手紧握镰刀——握的不是武器，是无数次危险夜晚养成的肌肉记忆
10-15秒：两人在框中框构图里对峙，各自一半明一半暗。参考《花样年华》周慕云与苏丽珍在楼梯口的经典双人构图——不让他们站在同一束光下
贯穿全程：城市低频噪音，第10秒起大提琴若隐若现
```

## 代词规则（强制）

- 每镜角色首次出现必须用全名，同一镜内再次提到可用代词
- **跨镜强制重写全名**：镜1写了小红，镜2第一句必须是「小红」

## 审阅输出格式

每场输出：戏型判定 + 镜头数 + 导演工作本

```
## 1-1 · 小泽公寓卧室
**戏型**：对峙/试探 | **镜数**：6镜
参照：《花样年华》中苏丽珍与周慕云在酒店房间摊牌的处理——克制、物件叙事、情绪在沉默里

1-1-1
[导演阐述段落]

1-1-2
[导演阐述段落]
```
