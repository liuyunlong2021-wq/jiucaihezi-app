# 短剧写作质量手册

> 融合从业多年积累的全部写作规则。**每一集的质量由这些规则决定。**

---

## 一、核心公式

```
每一集 = 建立预期 → 羞辱/拉扯 → 打破预期（反转）→ 留钩子
```

**核心思维**：每一句话、每一个动作、每一段描写，都服务于「建立预期并打破它」。没有例外。

---

## 二、单场戏中心事件

每场戏生成前，**必须内部确定一句话中心事件**。不对用户展示，只用于指导写作。

```
本场中心事件 = 谁，在什么压力下，被什么催化事件逼到什么选择，结果造成什么新变化。
```

**要求**：
- 只能有一个中心事件
- 所有台词、动作、冲突都围绕它
- 不写与中心事件无关的过渡
- 不在正文中展示这句话

---

## 三、节奏结构（起承转合）

| 段 | 占比 | 内容 |
|----|------|------|
| 起 | 10% | 冲突开场，制造噱头。直接切入矛盾，不要铺垫 |
| 承 | 80% | 羞辱主角、拉扯情绪，建立观众对「反击」的期待 |
| 转 | — | 反转打脸，情绪爆发。预期被打破 |
| 合 | 10% | 留钩子，铺垫下一集 |

**开篇铁律**：前 3 秒必须有冲突。不要背景介绍、不要铺垫、不要闪回。直接进入事件。

**事件推进**：抽取关键点，不写过渡。比如主角去酒店——不写打车、问路、找房间，直接写聚会场景的矛盾冲突。

---

## 四、台词铁律

### 每条台词必须有目的

- 塑造人物形象
- 推动剧情进程
- 加强情绪烈度
- 交代必要信息
- 递进人物关系

**如果一句台词没有以上任何目的——删掉。**

### 台词标准

- **短**：每句 ≤15 字。短狠准。每句只抛一个信息。
- **爆**：有爆发力。反派台词要粗俗难听，绿茶要光明正大作恶。能命令别请求，能撕破脸别委婉。
- **贴**：贴合人设。什么人说什么话。主角台词要符合当前压迫状态，不能无故装腔。
- **绕**：🚫 不要含蓄、不要委婉、不要绕弯子。短剧观众要的是直接的情绪冲击。
- **围**：每句都围绕当前冲突展开。不扯其他。

### 台词进阶

- 台词不够炸 → 先检查冲突有没有压够，不是加感叹号能解决的
- 一人连续说超过 3 句 → 拆：插入打断、动作、反击
- 能用一个词别说一句话

### 绝对禁止

- 🚫 心理活动（"他心想""他意识到""他觉得"）→ 转化为台词/动作/表情
- 🚫 大段独白（一人说超过 3 句就是问题）
- 🚫 解释性台词（"因为……所以……"这种句式是毒药）

---

## 五、情绪法则

### 情绪要强烈、不高级、粗俗

短剧不是话剧。观众要的不是优雅，是**爽**。

- 反派：坏得彻底，嚣张跋扈，张牙舞爪
- 绿茶：光明正大地作恶，毫不掩饰
- 泼妇：怎么难听怎么骂
- 女主角被欺辱：要让人看得揪心、愤怒

### 情绪拉扯技巧

1. **对比增强冲突**：一边幸福一边痛苦，一边高贵一边卑微
2. **先压低再爆发**：先让观众愤怒（承段 80%），再让观众爽（转段）
3. **情绪卡点**：角色喜怒哀乐、震惊、不可思议的表情 → 正是切钩子的时机

### 矛盾来源

- 角色 vs 角色（男主vs女主、女主vs女配、男主vs反派）
- 角色 vs 环境（社会地位、财富差距、身份悬殊）
- 角色 vs 自身（价值观冲突、性格缺陷导致的选择困境）

---

## 六、钩子法则

### 每集末尾必须有钩子

免费集**每一集**末尾留钩子，付费集最多两集留一个。

### 四种钩子类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 悬念钩子 | 事件将被解决而未解决 | 女主被陷害，证据指向女主，下一集揭晓 |
| 断章钩子 | 高潮中途卡断 | 打斗到一半、暧昧到一半、耳光举起来 |
| 出场钩子 | 关键时刻人物出现 | 女主被欺辱，男主推门而入 |
| 情绪钩子 | 强烈情绪定格 | 角色震惊/愤怒/不可思议的表情 |

### 钩子进阶

- 新人物登场（被男主打脸的反派叫来更大的靠山）
- 新事件发生（主角解决一个障碍，引发新的阻碍）
- 新事物展示（主角展示了某种能力/物品，引发他人觊觎）

---

## 七、冲突强度公式

### 核心公式

```
冲突强度 = 施力点的狠度 × 受力点的反馈强度
```

- **施力点越狠**：反派的手段越毒辣、羞辱越彻底 → 冲突越强
- **受力点反馈越强**：主角的反应越激烈、情绪越饱满 → 冲突越强

**示例**：
- 弱冲突：反派骂了主角一句，主角忍了
- 强冲突：反派当着所有人的面把主角母亲推下楼梯，主角冲上去拼命

### 危机感三件套

每个钩子/冲突场景至少包含一项：

| 要素 | 说明 | 示例 |
|------|------|------|
| 信息差 | 观众知道但角色不知道，或角色知道但其他角色不知道 | 观众看到反派下毒，主角不知 |
| 时间锁 | 必须在限定时间内解决，否则后果严重 | "天一亮你就得死" / "还有一炷香" |
| 生死局 | 赌上性命/全部身家/最重要的人 | 不赢就死 / 输了就失去一切 |

## 八、催化事件质量标准

### 好催化事件（至少满足 2 条）

- 触碰角色红线
- 制造两难选择
- 放大已有误会
- 引爆隐藏矛盾
- 打破角色安全状态
- 迫使角色立刻行动
- 让观众期待反击
- 解决一个问题，同时制造下一个问题

### 坏催化事件（避免）

- 只是有人来了
- 只是说了一句话
- 只是换场景
- 只交代设定
- 不迫使角色行动
- 力度不够，角色可以忽略

### 执行规则

- **施力点**：反派、环境、误会、时间限制、身份压迫、亲密关系背叛
- **受力点**：主角必须有明确损失、羞辱、危险、愤怒或反击冲动
- **反馈方式**：动作、表情、台词、沉默、反击——不用心理描写

---

## 九、单场结构

```
开场：催化事件直接砸下来（3秒内）
中段：羞辱、误会、威胁、压迫持续升级
转折：角色反应打破预期
结尾：新危险/新问题/新人物/新证据/强情绪定格
```

- 开场不要铺垫，催化事件 = 第一句话/第一个动作
- 中段每一轮冲突要比前一轮更狠
- 转折必须让观众意外但合理
- 结尾必须有钩子

## 十、写作技法

### 8 种剧情冲突写法

| 技法 | 说明 |
|------|------|
| 侮辱和背叛 | 被男友、闺蜜、亲近的人侮辱背叛 |
| 误会 | 被男主/关键人物无脑误会 |
| 暴力对待 | 身体暴力、情感暴力 |
| 肢体+情感冲突 | 虐身虐心 |
| 言语攻击 | 配角对主角的恶毒言语攻击 |
| 不信守承诺 | 人物出尔反尔 |
| 童年阴影 | 人物某种行为源于童年创伤 |
| 欺诈 | 观众上帝视角知道被骗，剧中人不知 |

### 反派阴险套路

- 强拆CP：抢主角女神/男神，利用主角性格缺陷制造矛盾
- 搞破坏：让主角自己拆自己的CP

### 20 个出圈反差人设

选择人设时参考（详见后方附录），核心是**表面 vs 内在的极致反差**。

## 十一、不可拍表达

以下词在剧本中**绝对禁止**。心理学动词 → 必须转化为可视化呈现。

```
发现自己 / 意识到 / 心想 / 觉得 / 明白了 / 知道了 / 想起
穿成了 / 穿书了 / 内心 / 脑海中 / 记忆中
决定（除非"决定战"）
```

**转译示例**：
- ❌ "林风发现自己被跟踪了" → ✅ △ 林风脚步一顿，余光扫向身后巷口。
- ❌ "她心想这次死定了" → ✅ 阿曼（咬牙）："这次跑不掉了。"
- ❌ "他意识到这是陷阱" → ✅ △ 他猛地停步。已经晚了——身后门闩落下。

---

## 十二、创作禁忌（10 条红线）

| # | 禁忌 | 说明 |
|----|------|------|
| 1 | 主线不明确 | 观众不知道这剧到底要讲什么 |
| 2 | 节奏慢 | 铺垫太多，切入太晚，10 集了还没进入正题 |
| 3 | 缺乏冲突 | 情节平淡，没有矛盾，没有吸睛点 |
| 4 | 铺垫过多 | 不要解释背景，用人物对话自然带出 |
| 5 | 配角太多 | 人物混乱，观众分不清谁是谁 |
| 6 | 逻辑不通 | 事件发展到最后圆不回来，割裂感 |
| 7 | 反派弱智 | 反派不能是傻子，要让主角赢得有价值 |
| 8 | 一帆风顺 | 主角不能轻松赢，要先被压制再逆袭 |
| 9 | 忽略拍摄成本 | 场景太复杂、特效太多、群演太多 → 拍不出来 |
| 10 | 大段独白 | 一个人说超过 3 句台词就是问题 |

### 删减规则（直接删除）

以下内容在短剧剧本中**一律删除**，不需要犹豫：

- 打车、问路、进门、倒水等无效过渡动作
- 背景解释（"事情的起因是这样的……"）
- 设定科普（"这个世界分为三个大陆……"）
- 人物内心分析（"他之所以这么做是因为……"）
- 不推动冲突的寒暄（"你好""好久不见""最近怎么样"）
- 和本场中心事件无关的支线

## 十三、新手问题修复表

| 问题 | 症状 | 修复方案 |
|------|------|----------|
| 台词软 | 对话像聊天，没有张力 | 每句加目的：羞辱/挑衅/威胁/揭露。删掉礼貌用语 |
| 动作散 | △叙述太多，画面感弱 | 每段△只写一个关键动作。删掉过渡动作（走路/坐下/转头）|
| 冲突弱 | 角色在争执但观众不揪心 | 升级施力点：加点肢体暴力或致命威胁。加危机感三件套 |
| 剧情突兀 | 前一场和后一场跳得太快 | 中间补一场过渡或用一句台词带过关键信息 |
| 定位不对 | 写得像小说/话剧/散文 | 回忆短剧核心：情绪、冲突、反转、钩子。不要优雅、不要含蓄 |
| 角色话多 | 一人说超过3句 | 砍。用动作或沉默替代。短剧观众不爱听长篇大论 |
| 开头太慢 | 前3秒没冲突 | 直接从矛盾中间切入。背景用对话自然带出，不要单独介绍 |
| 钩子弱 | 结尾平淡，观众不想点下一集 | 用四种钩子模板：悬念/断章/出场/情绪。重写最后一段 |

## 十四、格式规范

```
## 场X-Y
时间：[日/夜/黄昏]（+雨/雪/风如需）
场景：[室内/室外]+[具体地点]
人物：[[wiki/角色/A]] [[wiki/角色/B]] （有动作/台词的次要人物用甲乙丙标注）

△ [场景描述/动作描述]

角色名（情绪）："台词"
△ [动作描述]
```

- △ 叙述/动作/描写，每行独立
- 不同人物的台词各占一行
- 场景转换 = 新建一场
- 单集 500-1200 字（通常 1-3 场）
- 单场 200-400 字
- 付费卡点：第 8-10 集第一个，后续每 20 集一个，末尾标红 `【付费卡点】`
- OS = 内心独白，VO = 画外音
- 闪回标 `【闪回开始】…【闪回结束】`

## 十五、每集自检清单

- [ ] 开场 3 秒有冲突
- [ ] 每句台词都有目的（塑造/推动/强化/交代/递进）
- [ ] 承段（80%）充分羞辱/拉扯情绪
- [ ] 有反转打脸（预期被打破）
- [ ] 末尾有钩子（悬念/断章/出场/情绪）
- [ ] 无心理活动，无不可拍表达
- [ ] 反派不弱智，主角赢得有价值
- [ ] 无大段独白，无解释性台词
- [ ] 字数 200-400（单场）或 500-1200（单集）
- [ ] 运行 `scripts/validate_scene.py` 通过
