# 漫剧爽点库（30条）

> 这是漫剧/短剧中最常见的30种爽点模式。在创作剧本时，每3-5场戏埋入一个爽点，保持观众追读欲望。

## 身份反转类

1. 主角是一方霸主，但周围的人都把他当成菜鸟欺辱
4. 主角身份低微被所有人看不起，最后身份揭晓——他是世界的王者/狂拽富二代
9. 主角扮猪吃老虎，借别人之手打脸反派
16. 主角就是一个什么都不会的穷鬼/混混，但所有大佬对她都很恭敬
19. 聚会/宴会时遭到歧视羞辱，路过的大人物却纷纷对主角点头哈腰
27. 主角明面的身份是小人物，但很多人都知道遇到事情只有他能解决，他才是隐藏的大佬

## 实力碾压类

2. 主角是无敌存在，对待反派能动手绝不动口，不圣母心，以牙还牙
6. 反派以为主角放完大招就重伤不起，谁知更大的招还在后面，一击必杀
7. 反派以为主角是穷乡僻壤来的弱鸡，其实反派可望不可及的物品是主角发明的
13. 所有人都认为主角必死无疑，主角披着更厉害的马甲出现
14. 主角很强，但每次都表现得比配角好一点，配角以为能战胜，却被玩弄于股掌之中
23. 没人敢采撷的高岭之花对主角心动不已，而主角却不在乎
24. 主角参加比赛/大会时不被看好，结果却是降维打击，断层第一
25. 别人看来无法解决的惊天大难题，对主角来说小菜一碟
30. 主角在绝境中极限反杀

## 逆袭打脸类

5. 主角被原团队抛弃，自己创立团队比原来更厉害，原团队请求回归被拒
10. 主角被反派诬陷陷入绝境，所有人都远离时却来个大反转，主角反杀配角
12. 主角重生前被折磨，重生后彻底扭转命运
17. 对于帮助过自己的小人物，主角会以让小人物吃惊的礼物作为报答
18. 主角能够在短时间内完成别人不可能完成的任务
28. 主角做出一件惊天动地的大事件，对他来说却是一件平常不过的事情
29. 主角以一人之力收复整个帮派，并解决悬而未绝的难题

## 特殊设定类

3. 外人看来很普通的物品/武器，在主角手里就是宝藏/神器
8. 主角有幸运体质，不管遇到什么事都能逢凶化吉
11. 重生过去改变命运，改变自己遗憾的人生
15. 反派让美女用美人计陷害主角，美女却爱上主角/倾佩主角魅力而倒戈
20. 其他人害怕、闻风丧胆的人物，却是主角看不起的手下败将
21. 能够掌控世界的大人物，实际上是主角的傀儡
22. 主角维护自己人，不管那个人是多小的小人物，都能给予让反派无法企及的帮助
26. 主角身边的朋友都是各个领域的大佬，绝不结交废物，反派想要巴结都巴结不上
