# 导演场景节奏档案模板

> 阶段 1.5 生成节奏档案时读取。按此模板从训练数据中提取导演面对不同戏型的节奏处理方法。

## 档案结构

```
## 导演 · 作品 · 场景节奏档案

### 戏型1：XXX（参考该作品中哪几场戏）
- 镜头数范围：X-Y镜/场
- 切镜密度：秒/镜
- 手法：平行剪辑/长镜/正反打/蒙太奇/白描……
- 节奏特征：快→慢→定格？情绪爆发点在沉默还是在动作？
- 镜头运动习惯：推轨/固定/手持/摇镜……
- 景别组合：远景→中景→特写？或中近景为主？
- 光影与空间特征：光从哪来、色温、空间感
- 标志性处理：这个导演拍这类戏最独特的做法

### 戏型2：XXX
……
```

## 常见戏型分类

至少覆盖以下，根据导演作品调整：

- **武打/动作/追逐**：群战还是单挑？碎切还是长镜？
- **对峙/谈判/博弈**：正反打节奏、沉默占比、物件特写穿插
- **爱情/暧昧/分手**：不同阶段（初遇/热恋/分离）的镜头密度差别
- **孤独/独角戏**：镜头跟不跟人物、留白多少、环境音比例
- **群像/多人场景**：多人调度方式、谁的主视角
- **悬疑/揭示/reveal**：信息揭露的节奏、镜头欺骗观众的手法
- **喜剧/荒诞**：喜剧节拍在镜头里怎么落
- **史诗/大场面**：远景比例、是否用航拍/大摇臂

## 使用方式

档案生成后，阶段 2 逐场匹配时直接查表——不需要重新推理这个导演的风格，只需要判定当前场是什么戏型、处于什么阶段。
