# 导演作品参考库（Director Style Bank）

本库是 `JC-剧本转分镜` 的**格式与一致性锚点**。真正有价值的是 LLM 从训练数据中对这些导演的深度知识——本库只提醒有哪些可参考的维度。

字段约定：
- `style_design` 该作品标志性光影语言，格式 `[布光] + [色调] + [反差] + [质感]`。
- `pacing_design` 该作品真实节奏与切镜密度（秒/镜、碎切或长镜），阶段 1.5 节奏档案的起点。
- `机位习惯` 阶段 1.5 的镜头运动/景别偏好的参考锚点。
- `适用` 适合什么剧本类型。

### 王家卫《花样年华》
- 方向：情绪作者型
- ratio_design：9:16 / 16:9
- style_design：[框中框暖橘台灯] + [霓虹青冷边] + [中高反差] + [浓艳胶片质感]
- pacing_design：慢推慢拉，2-4秒一镜，情绪特写与留白优先
- 机位习惯：框中框构图、长焦空间压缩、缓慢推轨、人物常置于门框窗格内、旗袍色彩主导
- 适用：都市暧昧、暗恋、时代怀旧、双人情绪戏

## 使用提示

阶段 1 推荐时，从以上方向各取一个，保证 A/B/C 覆盖不同方向（如「情绪作者型 + 写实温情型 + 冷峻惊悚型」），让用户有明显取舍。若剧本明显偏某类，仍应给出差异化的三个选项，而非三个近亲。
