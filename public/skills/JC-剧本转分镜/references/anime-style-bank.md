# 动漫作品参考库（Anime Style Bank）

本库是 `JC-剧本转分镜` 的**格式与一致性锚点**。真正有价值的是 LLM 从训练数据中对这些动画的深度知识——本库只提醒有哪些可参考的维度。

字段约定：
- `style_design` 该作品标志性上色与光影语言，格式 `[上色/布光] + [色调] + [反差] + [质感]`。
- `pacing_design` 该作品分镜节奏与切镜密度（秒/镜、碎切或长镜），阶段 1.5 节奏档案的起点。
- `演出特征` / `技法要点` 阶段 1.5 的镜头运动/景别偏好的参考锚点。
- `适用` 适合什么剧本类型。

## 现代商业动画（日系 / 国漫 3D）

### 最终幻想（如《最终幻想VII：圣子降临》/ 野末武志 · Square Enix CG）
- 方向：写实 CG 电影型
- ratio_design：16:9 / 2.39:1
- style_design：[CGI 电影光] + [冷蓝高光 + 暖橙补光] + [高反差] + [照片级 CG 质感]
- pacing_design：电影感慢镜与高速战斗交替，3-6秒一镜，特效大招长镜优先
- 演出特征：CG 渲染、体积粒子特效、镜头光晕、英雄落地慢镜、刀光拖影
- 适用：奇幻、科幻、CG 电影感、史诗战斗

### 海贼王（尾田荣一郎原作 / Toei 动画）
- 方向：热血明亮型
- ratio_design：16:9
- style_design：[平涂亮色] + [高饱和暖调（草帽红 / 海蓝）] + [低反差] + [电视动画 cel 质感]
- pacing_design：中快，2-4秒一镜，觉醒/变身长镜，热血演出优先
- 演出特征：速度线、集中线、夸张表情、橡皮伸缩、霸气黑化、海楼石冷光
- 适用：少年热血、冒险、喜剧、群像

## 美影厂美术片（上海美术电影制片厂）

> 美影厂是中国动画学派代表，技法与日系 cel 完全不同。下面按「一种技法 = 一个可选项」拆开，方便阶段 1 给出差异化选项。竖屏短剧可统一改为 9:16；原始比例多为 4:3 / 1.37:1。

### 美影厂《大闹天宫》工笔重彩（明亮华美变体）
- 方向：民族工笔重彩 · 喜庆神话
- ratio_design：4:3 / 1.37:1（竖屏改 9:16）
- style_design：[铁线描平涂] + [金红主调 + 祥云暖色] + [平光无渐变] + [工笔重彩质感]
- pacing_design：戏曲程式中快，3-6秒一镜，亮相定格与舞台调度优先
- 技法要点：铁线描黑勾（粗细均匀无压感）、单线平涂无渐变、矿物颜料（金/朱砂/石青/石绿/藤黄）、京剧脸谱化造型、散点透视、装饰飘带祥云、敦煌壁画式平面构图
- 适用：中国神话、天庭、龙宫、西游降妖、喜庆庙堂

### 美影厂《哪吒闹海》工笔重彩（悲壮沉郁变体）
- 方向：民族工笔重彩 · 悲剧英雄
- ratio_design：4:3 / 1.37:1（竖屏改 9:16）
- style_design：[铁线描平涂] + [深青/银白/朱红大色块] + [平光无渐变] + [工笔重彩质感]
- pacing_design：悲壮长镜，4-8秒一镜，海水江崖与狂风暴雨气氛优先
- 技法要点：同工笔重彩体系，但色调压暗、大色块对比强烈、海水江崖传统纹样、牺牲戏悲情留白
- 适用：悲剧英雄、反抗、神话、父子情

### 美影厂《山水情》水墨（文人泼墨变体）

## 使用提示

阶段 1（动漫侧）推荐时，同样从以上方向各取一个（如「热血明亮 + 酷炫速度 + 美影厂工笔重彩」），保证 A/B/C 覆盖不同方向，让用户有明显取舍。动漫侧也必须具体到作品/风格，不能只给「日系」「热血」空词：
- 现代动画：`director` 写监督 / 动画公司 / 原作，`work` 写具体作品（如《火影忍者》《凡人修仙传》）。
- 美影厂：`work` 写「风格 + 代表作」（如《大闹天宫》工笔重彩 / 《九色鹿》敦煌壁画），让用户清楚选的是哪一种民族美术片技法。
