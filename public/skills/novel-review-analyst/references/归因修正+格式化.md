# SKILL-05: 归因修正+格式化+终检器 (Attribution & Formatter)

> 🩺 "不仅要发现病灶，还要开对药方。最后，出一份体检报告。"

---

## 核心职责

1. **归因修正**：避免将问题简单归咎于"文笔不好"，深入挖掘结构性原因。
2. **报告生成**：输出标准化的复盘诊断报告。

---

## 归因修正表 (Attribution Correction)

**❌ 错误归因** vs **✅ 正确归因**

| 现象 | 错误归因 (表面) | 正确归因 (本质) | 建议方案 |
|:---|:---|:---|:---|
| **读者说"水"** | 我写的太细了 | 缺乏信息增量 / 缺乏冲突 | 删减日常，增加转折 |
| **读者说"平"** | 我文笔不够华丽 | 节奏像直线，没有起伏 | 使用"心电图"模型调整节奏 |
| **读者说"虐"** | 读者心理承受力差 | 连压时间过长，缺乏希望 | 缩短压抑期，给反击暗示 |
| **读者说"崩"** | 我没写好人设 | 剧情线强行扭曲了人设 | 检查四线对齐，修正逻辑 |
| **数据不好** | 这届读者不行 | 开篇没钩子 / 题材不热 | 检查前三章黄金开局 |

## 标准复盘报告模板

```markdown
# 🩺 章节复盘诊断书：第X-Y章

## 📊 核心指标
- **情绪曲线**：[📈上升 / 📉下降 / 〰️波动 / ➖直线]
- **节奏评价**：[紧凑 / 适中 / 拖沓]
- **四线状态**：
  - 剧情：[✅ 推进 / ⏸ 停滞]
  - 感情：[✅ 升温 / ⏸ 僵持]

## 🚨 风险预警
- **连压预警**：[无 / ⚠️ 连续3章憋屈]
- **读者视角**：[✅ 信息清晰 / ❌ 存在上帝视角盲区]
- **卖点密度**：[⭐⭐⭐]

## 💡 深度诊断 (SKILL-01 & 02)
- **问题点**：[例如：第15章虽然是高潮，但只有剧情推进，男女主全程零交流，感情线缺失。]
- **归因**：[结构失衡，过于侧重打脸而忽略了互动。]

## 💊 修改建议
1. **[紧急]** 在第15章结尾增加一段男主为女主包扎伤口的戏份（补感情线）。
2. **[优化]** 删减第14章的心理描写，加快进入高潮的节奏。
3. **[保持]** 反派的智商在线，继续保持。

```

## 终检清单 (Final Checklist)

- [ ] 诊断是否基于"读者视角"而非作者辩解？
- [ ] 建议是否具体可执行（不要说"提升文笔"，要说"删除这300字"）？
- [ ] 语气是否客观、理性但有建设性？
