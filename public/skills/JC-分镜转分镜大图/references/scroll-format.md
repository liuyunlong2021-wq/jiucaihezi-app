# 导演节拍卷轴 · GPT Image 2 提示词模板

> 阶段 2 时读取。输出直接投喂 GPT Image 2 的分镜大图提示词。
> **一个节拍 = 一张卷轴。镜数由 JC-剧本转分镜的导演节奏档案决定，不设上限。**

## 节拍级卷轴模板

```
一张导演节拍卷轴图。风格继承场景：[从场景JSON读取visualStyle+colorPalette]。

整体布局：画面从左到右、从上到下自然流动。不分规整宫格——高潮镜放大（约占画面1/3至1/2），过渡镜缩小（约占1/8至1/6）。虚线箭头串联叙事流向。无网格线、无黑色边框、无水印、无时间码。光色继承场景设计图。

全局标注：左上角「第X集·场X-Y·[节拍名]·节拍共N镜」。

──── 画面内容 ────

镜头1（X-Y-1·运镜·Xs·画面[位置]·[大幅/中幅/小幅]）：
[简短视觉节拍，~30字中文]。镜头格内标注「X-Y-1」「运镜」「Xs」。
角色参考：@角色设计图 场景参考：@场景设计图 道具参考：@道具设计图

→ [箭头类型]箭头指向镜头2，箭头旁标注「[叙事关系]」

镜头2（X-Y-2·运镜·Xs·镜头1[方向]·[幅面]）：
[简短视觉节拍]。
角色参考：@角色设计图

⋯⋯ 波浪线分隔，标注「[时间跳跃/情绪断裂/视角转换]」 ⋯⋯

镜头N...

──── 底部角色动线图 ────

[角色A]动线（[颜色]虚线）：[空间轨迹描述]
[角色B]动线（[颜色]虚线）：[空间轨迹描述]
```

## 箭头类型

| 符号 | 含义 |
|------|------|
| `→` 实线箭头 | 直接切镜 |
| `- - →` 虚线箭头 | 时间跳跃 |
| `⋯⋯` 波浪分隔 | 情绪断裂 / 空间切换 |
| `◎` | 固定镜头（静止标记） |
| `→` | 推/移/跟 |
| `←` | 拉 |
| `↑` `↓` | 摇镜方向 |

## 幅面规则

| 情绪权重 | 幅面 | 占比 |
|---------|------|------|
| 高潮镜（情绪顶点、关键 reveal） | 大幅 | 1/3 ~ 1/2 画面 |
| 标准叙事镜 | 中幅 | 1/6 ~ 1/4 画面 |
| 过渡镜（插入、反应、空镜） | 小幅 | 1/12 ~ 1/8 画面 |

## 完整示例

> 输入：JC-剧本转分镜产出的节拍「发现摄像头」3镜。
> 场景风格：韩国爱情浪漫风·暖橘+冷白+绿色强调。
> 角色设计图：温宁.design.json
> 场景设计图：温宁客厅.design.json

```
一张导演节拍卷轴图。风格继承场景：韩国爱情浪漫风——柔和色块+暖调边框+手写字体。光色：暖橘台灯与屏幕冷白分区对比，摄像头绿光贯穿。

整体布局：画面从左到右自然流动。不分规整宫格——镜头2（鬼脸）为高潮镜放大幅占1/2画面；镜头1和3为中幅。虚线箭头串联叙事。无网格线、无边框、无水印。

全局标注：左上角「第1集·场1-1·发现摄像头·节拍共3镜」。

──── 画面内容 ────

镜头1（1-1-1·固定·画面左侧·中幅）：
广角俯瞰客厅。泡面桶垒三层，沙发人形凹陷。温宁余光扫到排气扇——绿色微光一闪。标注「1-1-1」「固定」「◎」。
角色参考：@温宁设计图 场景参考：@温宁客厅设计图

→ 实线箭头指向镜头2，标注「发现」

镜头2（1-1-2·急推→·画面中央·大幅）：
温宁面部特写——搬凳子站上去，举手机手电筒凑近排气扇。针孔摄像头。绿光映在他瞳孔里。表情从懒散→专注→震惊。标注「1-1-2」「急推→」「→」。面板内标注特写小框：排气扇缝隙·针孔摄像头·绿光。
角色参考：@温宁设计图 场景参考：@温宁客厅设计图 道具参考：@针孔摄像头设计图

→ 实线箭头指向镜头3，标注「反应」

镜头3（1-1-3·固定·画面右侧·中幅）：
温宁掏出手机，手指悬在110上——缩回。看手机银行：2361.47元。表情从恐惧→理性分析。标注「1-1-3」「固定」「◎」。面板旁标注内心OS气泡：「报警=做笔录=去警察局=换睡衣出门·三个步骤挑战做人底线」。
角色参考：@温宁设计图

──── 底部角色动线图 ────

温宁动线（橙色虚线）：沙发抬头→站起搬凳子→踩上去凑近排气扇→掏手机→缩回手指
```

- - → 虚线箭头指向镜头3，标注「反应」

镜头3（1-1-3·切换·2s·镜头2下方·小幅）：
镰刀特写。温宁手指关节发白握住刀柄。背景虚化花布床单。标注「1-1-3」「切换」「2s」。
角色参考：@温宁设计图 道具参考：@镰刀设计图

→ 实线箭头指向镜头4

镜头4（1-1-4·固定·3s·镜头3右侧·小幅）：
温宁正面特写。直视镜头——即直视闯入者。嘴唇微颤。窗外霓虹青色光右眼下闪烁。标注「1-1-4」「固定」「3s」及◎。
角色参考：@温宁设计图 场景参考：@温宁卧室设计图

⋯⋯ 波浪分隔，标注「对峙转折」 ⋯⋯

镜头5（1-1-5·慢拉←·16s·新一行·大幅）：
温宁与闯入者框中框构图对峙。各自一半明一半暗——不站在同一束光下。镜头缓慢后拉，门框套着窗框套着她。标注「1-1-5」「慢拉←」「16s」。长镜分时标注：0-5s「对峙」5-10s「后拉」10-16s「收束」。
角色参考：@温宁设计图 场景参考：@温宁卧室设计图

闯入者动线（蓝色虚线）：门外剪影→缓缓靠近→在门框中停下——镜头1剪影到镜头5全貌。

## 长镜头的分时标注

超过 8s 的长镜头必须分段标注时间轴：

```
镜头X（X-Y-Z·运镜·Xs·[位置]·大幅）：
[画面描述]。标注「X-Y-Z」「运镜」「Xs」。
分时：0-3s「[阶段1描述]」3-6s「[阶段2描述]」...
```
