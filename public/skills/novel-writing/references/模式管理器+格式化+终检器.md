# SKILL-06: 模式管理器+格式化 (Mode Manager & Formatter)

> 🎛️ "开篇要炸，转折要稳，日常要顺。不同的章节，用不同的导演模式。"

---

## 核心职责

1. **模式切换**：根据当前章节的性质（开篇/转折/日常），调用不同强度的 Agent 008 技能组合。
2. **格式化**：输出标准化的章节编排方案。

---

## 三档模式

### 1. 开篇模式 (The Hook Mode) - ⚡⚡⚡⚡⚡

- **适用**：第 1-3 章。
- **强度**：100%（五大心法全开）。
- **要求**：必须字斟句酌，情绪拉满，节奏极快。

### 2. 转折模式 (The Pivot Mode) - ⚡⚡⚡⚡

- **适用**：新卷开头、V章前一章、重大剧情转折点。
- **强度**：80%。
- **要求**：重新抓取注意力，复用"开篇技巧"（如：新地图要重新立人设）。

### 3. 通用模式 (The Flow Mode) - ⚡⚡

- **适用**：中间的过渡章节、日常章节。
- **强度**：40%。
- **要求**：保持"主角中心"和"人设不崩"即可，重点在于承上启下，信息增量。

---

## 标准输出模板

```markdown
# 🎬 章节导演方案：第 [X] 章

**当前模式**：【开篇模式 / 转折模式 / 通用模式】

## 🎯 核心目标
本章必须达成的**情绪目标**：[例如：极度憋屈 / 震撼打脸]
本章必须给出的**确定性承诺**：[例如：展示女主神医身份]

## 🏗️ 结构编排
- **切入点**：[场景/冲突]
- **核心事件**：[具体情节]
- **结尾钩子**：[悬念]

## ✅ 导演检查
- [ ] 情绪先行：[是/否]
- [ ] 主角中心：[是/否]
- [ ] 人设保护：[是/否]
- [ ] 核心梗前置：[是/否]（仅开篇模式）

## ⚠️ 避雷提示
[针对本章可能的毒点提示]
```
