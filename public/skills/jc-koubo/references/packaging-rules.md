# 包装规则

包装是独立于常驻字幕和高光文案的第三条轨道。每张高光最多两件。

| 内容信号   | 组件               |
| ---------- | ------------------ |
| 问题、歧义 | `question_sticker` |
| 数字、比例 | `data_badge`       |
| 反差、纠错 | `contrast_arrow`   |
| 步骤、顺序 | `step_badge`       |
| 案例、人物 | `case_tag`         |
| 无强信号   | `keyword_spark`    |

默认全放在上方安全区，不进入常驻字幕区域，不进入人物中间三分之一。
