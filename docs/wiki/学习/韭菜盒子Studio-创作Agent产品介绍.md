## 04｜卖点一：一个账号，全模型

**页标题**  
一个账号，中外模型全开

**主句**  
登录一次。  
写、画、拍、改，全在一个工作台。

**四点**

- 选型自由：哪个模型强，就用哪个  
- 路径统一：不离开当前项目  
- 资产不断：结果回到同一文件  
- 团队省事：不用装齐全网工具

**底句**  
模型可以很多。入口只需要一个。

---

## 05｜痛点：AI 一聪明，就爱忘

**页标题**  
今天创作人的真实死局

**左卡：编程 Agent**  
强，但贵  
路径重，Token 烧得快  
不像创作者

**右卡：直接聊大模型**  
快，但飘  
聊完就淡  
角色设定全靠重喂

**结论**  
要么笨拙地贵。  
要么轻巧地忘。

## 06｜卖点二：第二大脑，永不失忆

**页标题**  
第二大脑：让项目真相住进文件

**五层**

| 层 | 一句话 |
| --- | --- |
| 项目 Wiki | 角色 / 场景 / 道具 / 剧情，单一真相源 |
| 热记忆 hot | 每次开聊，先对齐主线 |
| 事实账本 | 说过什么、改过什么，可追溯 |
| 巡检修正 | 查穿帮、漏改、断链 |
| 对话转 Wiki | 聊过的不会白聊 |

**底句**  
不赌上下文。  
把细节写进项目。

## 14｜对照表

**页标题**  
别家 vs 韭菜盒子
