# 韭菜盒子 Studio — 产品介绍

> **一句话**：韭菜盒子 Studio 是一个本地优先的纯手动 AI 工作台。用户一键登录即可使用 Claude、GPT、Grok 等全部顶尖模型，自由组合 Skill、知识库、创作工具，无需折腾配置、不用翻墙、不写代码。

---

## 一、产品定位

**AI 工作台，不是 AI Agent。** 韭菜盒子的核心理念是"用户主导，AI 执行"——用户选择模型、选择 Skill、选择工具、发出指令，AI 按显式配置精准执行。不搞黑盒自动化，不替用户做决定。

| 你是... | 你能用韭菜盒子做什么 |
|---------|---------------------|
| 🧑‍💻 普通用户 | 打开即用，一键登录就能和顶尖模型对话、生图、做视频 |
| 📝 内容创作者 | 用 Skill 系统调用专业能力包（律师、编剧、设计师...），搭知识库防失忆 |
| 🏗️ 开发者 | 本地项目读写、命令执行、代码生成，OpenCode 文/武双模式 |
| 🎨 设计师 | 创作面板一站式生图、生视频、生音频，13 个媒体模型随意切换 |

## 二、核心特点

### 2.1 零门槛，一键登录

```
下载 → 一键登录 → 即刻使用全部顶尖模型
```

- **不需要 API Key**：账号登录即用，内置 NewAPI 中转，统一鉴权。
- **不需要翻墙**：本地桌面应用，直连国内加速节点。
- **不需要配置**：模型列表自动同步，一键切换 Claude / GPT / Grok / Gemini 等 30+ 模型。
- **一键抄配置**：登录后点一下，复制全套 API 信息到剪贴板，在任何 AI 客户端使用。

### 2.2 全模型自由切换

内置 30+ 顶尖模型，一个界面随意切换：

| 类别 | 代表模型 |
|------|---------|
| 文本对话 | Claude Sonnet 4.6、GPT-4o、Grok-4、Gemini 2.5 Pro、DeepSeek V3 |
| 图片生成 | GPT Image 2、Nano Banana、FLUX Klein 9B |
| 视频生成 | Veo 3、Grok Video、Seedance 2.0 |
| 音频生成 | Suno V5、RH 声音 |

- **模型感知**：自动检测模型能力（vision / text-only），图片自动适配格式，非视觉模型贴图自动转文字描述。
- **Token 水位计**：实时显示上下文用量，三色预警，不丢上下文。

### 2.3 Skill 系统 — 给 AI 装上专业大脑

Skill 是官方 Anthropic Skill 标准的能力包。每个 Skill 是一套 `SKILL.md` + 参考资料 + 脚本，让 AI 瞬间获得专业领域能力。

| 分类 | 内置 Skill |
|------|-----------|
| 专业领域 | 律师工作台 |
| 内容创作 | 漫剧剧本生成器 |
| 创意设计 | 品牌指南、前端设计、海报生成、算法艺术 |
| 开发技术 | Claude API、MCP 构建器、Skill 创建器 |
| 企业沟通 | 文档协作、内部通讯 |
| 文档技能 | Word / PDF / PPT / Excel 生成与转换 |
| 知识管理 | 知识库架构师 |

- **中央 Skill 库**：开箱即用 20+ 官方 Skill，GitHub 一键导入社区 Skill。
- **用户自建**：对话、编辑器、知识库里提炼经验，自动生成专属 Skill。
- **纯手动**：用户选择 Skill → AI 按 SKILL.md 执行，不自动瞎选。

### 2.4 知识库 — 给 AI 装上长期记忆

知识库解决 AI "聊完就忘"的核心痛点。用户上传资料 → 系统整理为结构化知识页 → 对话时自动召回。

```
用户上传资料 → raw/ 原始素材
         ↓ 用户手动触发整理
     wiki/ 结构化知识页
         ↓ 对话时自动召回
   LLM 获得前情提要，续写不失忆
```

**三层架构**：
- **raw/** — 原始素材，用户拖拽上传，不改动。
- **wiki/** — 整理后的知识页，可检索、可引用。
- **CLAUDE.md** — 记忆锚点，定义知识库规则。

**安全底线**：知识库只接受用户手动添加，AI 禁止自动写入，杜绝幻觉污染。

**开箱模板**：律师案件库 / 小说知识库 / 漫剧剧本项目，一键创建。

### 2.5 创作面板 — 一站式 AIGC 工厂

不需要切换多个工具，韭菜盒子内置全媒体生成：

- **图片**：文生图、图生图、比例/分辨率/风格自由控制。
- **视频**：Veo、Grok Video、Seedance 三大引擎，比例/时长可调。
- **音频**：Suno 音乐生成，标题/标签/MV 模式。
- **画廊**：所有作品自动归档，一键下载、复制链接、灯箱预览。

**13 个媒体模型**，统一鉴权，统一入口，生成结果自动本地保存。

### 2.6 本地工具 — 桌面端的真正力量

桌面应用不只是网页套壳。韭菜盒子直接调用操作系统能力：

| 能力 | 说明 |
|------|------|
| 📄 文档转换 | Office / PDF 转 Markdown，本地处理不泄密 |
| 🌐 浏览器控制 | 可见 Chrome 搜索、打开网页、截图、点击 |
| 💻 项目读写 | 在用户选择的项目目录内读写文件、搜索、执行命令 |
| 🎬 媒体处理 | ffmpeg 视频处理、yt-dlp 网页媒体采集 |
| 🔌 OpenCode | 文模式/武模式双引擎，项目级 AI 协作 |

### 2.7 双端覆盖

| 平台 | 形态 | 特点 |
|------|------|------|
| 🖥️ 桌面端 | macOS / Windows 原生应用 | 全功能：本地工具 + OpenCode 文/武模式 + 直连模式 |
| 🌐 Web 端 | 浏览器访问 | 轻量直连模式，随时随地打开即用 |

双端共享核心体验：模型、Skill、创作面板、知识库，视觉和交互保持一致。

## 三、技术亮点

- **本地优先**：数据存本地 SQLite，Session Token 0600 权限文件存储，不上传用户数据。
- **流式响应**：SSE 实时流式输出，逐字显示，体验对标 ChatGPT。
- **安全架构**：Rust + Tauri 桌面壳，权限最小化，路径穿越防护，外部链接走系统浏览器。
- **离线降级**：存储层故障时自动降级到 localStorage，UI 不白屏。
- **三平台发布**：macOS ARM / macOS Intel / Windows x64 三平台 CI 自动构建。

## 四、与同类产品的区别

| 对比维度 | ChatGPT / Claude 网页版 | 其他 AI 桌面客户端 | 韭菜盒子 Studio |
|---------|------------------------|-------------------|----------------|
| 模型数量 | 单一厂商 | 需手动配 Key | 30+ 模型一键切换 |
| 使用门槛 | 需翻墙 + 付费 | 需自行申请 API Key | 一键登录即用 |
| 知识库 | 无或有限 | 通常无 | 三层架构 + 防幻觉污染 |
| Skill 系统 | 无 | 无 | 官方 Anthropic Skill 标准 |
| 本地工具 | 无 | 有限 | 浏览器控制 + 项目读写 + 媒体处理 |
| 媒体生成 | 仅图片 | 需额外配置 | 图/视频/音频一站式 |
| 数据隐私 | 全在云端 | 取决于实现 | 本地 SQLite，密钥 0600 权限 |
| 双端覆盖 | 仅 Web | 仅桌面 | 桌面 + Web 双端 |
