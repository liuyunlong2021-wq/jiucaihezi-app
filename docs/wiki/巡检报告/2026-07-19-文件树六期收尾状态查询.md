## 最终架构事实

[[开发/文件系统/文件树五期编辑区与项目文档统一SDD]] 定义的最终合同是：

```text
项目文件 / 项目素材
  -> ProjectResource + ProjectFileService
  -> ProjectResourceChange
  -> 文件树（可视化入口）/ 编辑区 / 创作面板 / 画布（消费者）
```

项目资源服务是唯一文件总管；编辑区、创作面板和画布不得新增路径拼接、平台文件 API 或独立项目存储规则。
