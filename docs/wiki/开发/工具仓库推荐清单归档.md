## 产品处理决定

- App 不再展示、搜索、安装、卸载或扫描这 19 项。
- SuperMemory 和 Superpowers 已作为“插件”迁入 `设置 → 插件`，但不再由本清单供数；插件页只保留自己的两个固定推荐。
- 未来需要推荐内容时，在搭子 Studio 或独立网站展示这份清单，链接直接指向原项目。

## 必须保留的运行时能力

这些不是工具仓库，不能随着推荐体系删除：

- OpenCode 二进制定位：文武模式的核心。
- MCP 的本地命令查找：启动 stdio MCP 所需。
- 创作媒体处理对 `ffmpeg`、`ffprobe`、`whisper`、Python 的本机查找：用户自行安装后，创作模式仍可调用。

运行时只从系统 `PATH` 和 `~/.jiucaihezi/tools/` 查找程序，不再显示“已安装”状态，也不再提供安装入口。
