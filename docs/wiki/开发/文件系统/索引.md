## 唯一事实源

```text
项目文件 / 项目素材
  -> ProjectResource + ProjectFileService
  -> ProjectResourceChange
  -> 文件树 / 编辑区 / 创作面板 / 画布
```

`ProjectFileService` 是唯一项目文件总管；文件树只是可视化入口。消费者不得自行拼项目路径、调用平台文件 API 或维护独立项目资源存储。
